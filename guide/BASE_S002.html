[I want to configure the starting party]

[Goal]
　Configure the sample game so that Wolfar and Yuichi start
　already in a party together from the beginning.

[1]
　Click the Changeable Database
　to open the window shown on the right.
　　
[2]　Select "Party Info" from the "Type" list.
[3]　On the right side you will see entries labeled Member 1–6.
The characters selected here are the
party members present at game start.
[4]　Click a member slot to choose which character to assign.
Try adding or swapping characters.
Here we added "Yuichi".

※ The characters you can choose here are the ones configured
　in "Player Character Status".
　→ See: [I want to add new player characters]
⇒
[5]　Run Test Play and
　　　Yuichi is now following along successfully.
[Side Note]

※ If you leave Member 1 as "None" and use the Basic System menu, errors may occur.
　　If you want an invisible leader or unusual setups, consider creating a transparent character.

※ Having the same player character appear more than once as a member causes problems.

　　Example: If all members are set to "Yuichi", after combat ends all Yuichis will have
　　the HP, SP, and status ailments of the last Yuichi in the party.
　　Equipping one Yuichi will give the same equipment to all other Yuichis too.

　　If you want multiple characters with the same appearance, create separate player characters for each.
