[How do I create a new map?]

[Goal]

　Create a new map called "test" (filename: test.mps),
　draw "RPG" on it using grass chips as shown below, and save it.

　

　[Warning!]
　　Note: completing this step alone will not allow the player to reach this map.
　　You also need to complete 'I want to move the player character to a different map'.

[1]　Click Create New Map .
[2]　The "Create New Map" window
　　　shown on the right will appear.
　
[3]
　For now, just fill in the parts
　highlighted by the red box.

　Set the filename to "test( .mps )"
　and the map name to "Test".

　When done, press "OK".


　※Make sure the filename does not
　　conflict with other map files.
　　If you accidentally use the same filename,
　　the original data will be overwritten and lost.
　　If "test" doesn't work, try "test2" or "test3".
　
[4]　A new map has been created!


　　It's all blue.
　　In the game, this blue area
　　is displayed as black.
[5]　Since we're here, let's try
　　　drawing on the map.

　　　Click Layer 1 to open
　　　the map chip window on the right.

　
[6]
　Click the grass chip in the map chip window, then
　click in the main window (the blue area)
　to place the grass chip.

　Now try drawing on the map like a painting!
[7]　
　We drew freely with grass chips
　to write "RPG!" on the map.

　When you're done, click Overwrite Save
　to save the map.


　Your first map is complete!
　Great work!
　
Place chips like you're painting a picture.
[Extra]　I placed a chip in the wrong spot!
　　
　You tried to write "RPG!" but placed
　a chip in the wrong spot — that happens!

　In that case, select the top-left chip
　in the map chip window
　and click the mistake to erase the chip.

　The top-left chip acts as an eraser.

　※If you want to be able to move the player to this map,
　　 proceed to 'I want to move the player character to a different map'.



[Side Note]

　Even after doing all the steps above,
　the Map Selector window
　may not yet reflect the new map.
　Click the "Update Data" button
　to reload the map names.

　The map name "4: Test" will appear
　as shown on the right.
