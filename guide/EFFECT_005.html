【I want to change the font in a message】

[Goal]
In WOLF RPG Editor, you can use any
font you like anywhere in the game.

For example, you could make keywords like "Forest of Confusion" or "Demon King's Castle"
stand out by displaying them in a slightly different font.
Or have a magic user speak incantations using
a special character font during a magic-use event.
Another idea: display text in an unreadable symbolic font,
and only show it in a readable font when the player
has the item "Ancient Manuscript".
Being able to change fonts opens up many possibilities.

This time, we will learn how to change the font in a message
by using the Show Message Event Command.

Note that what we introduce here is how to use a font that has already been
registered for use in WOLF RPG Editor.
For how to register a font,
see "I want to use a font I like in my game".

【Steps】

1. Changing the font
・Create an event at a suitable location in "Sample Map A"
　and set the chicken image.
・Click "■ Show Command Input Window ■" in the lower right
　of the Event window.
・Change the tab in the Event Command Input window
　to "1 Show Message".
　Then enter the content:
　\font[1]Key A　\font[0]Key A
　and press the "input" button.
　　The "Key A" part doesn't really matter;
　　what's important is "\font[(number)]".
　　This is a special character command that switches the font
　　for subsequent text. The number corresponds to
　　the registration number of a font registered in
　　"Game Basic Settings" (opened from "Game Settings")
　　under the item "Fonts Used (blank = 'MS Gothic')" (0～3).
　　There are Basic through Sub 3, with registration numbers 0, 1, 2, 3 from top to bottom.
　　By default, "MS Gothic" is in the basic font slot and
　　"Arial Black" is in Sub Font 1,
　　so text after "\font[1]" will be displayed in Arial Black,
　　and text after "\font[0]" will be displayed
　　in MS Gothic.
This is where they are registered

Enter it like this
2. Confirm
　Now Test Play and talk to the event.
　We entered "Key A" twice, but in the Arial Black font version,
　the character for "key" has been changed to a symbol resembling the letter R.
　This is because Arial Black is a font for Western languages and cannot
　display Japanese characters correctly. As a result, when you enter kanji or hiragana,
　symbols or French characters may appear.
　The "A" in the other one is correctly displayed, but you can clearly see
　differences in weight, height, and width.
　If it doesn't work correctly, review step 1.
　Good work!
That's the font difference

[Side Note]

The "basic font" specified by \font[0] is the same as the font used
when text is entered normally without any command.
Therefore, the use case for \font[0] is, as in this example,
when you want to return to the basic font after using a different font
in a string.

This time we introduced how to actually use a font that has already been
registered in Game Settings for use in WOLF RPG Editor.
The registration method is described in
"I want to use a font I like in my game",
so if you need that, please check there too.
