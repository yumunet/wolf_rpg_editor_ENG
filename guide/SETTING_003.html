【I want to change the choice window image】

　[Goal]

　By changing the window image, you can easily
　change the atmosphere of your game.

　This time, we will change the image that serves as the background when a choice is displayed to "CursorBase.png",
　and learn how to change system images.

　【Steps】
[1] Open the System DB

　Clicking System Database will display
　a window like the one on the right.
[2] Select the type to edit

　① Right after the window opens, the "Type" is set to "Map Settings".
　Click "10: Window Image".
　② There is a data list in the middle of the window.
　Select "1: × Choice Window Image" from this list.
[3] Set the image

　Next, set the image to be used for the window.
　Press the [File] button to the right of the image filename;
　this opens the folder containing assets.
　Select "CursorBase.png" and press the
　"OK" button at the bottom-right of the window.
　"SystemFile/CursorBase.png" has now been
　entered as the image filename. Press the
　"OK" or "Update" button at the bottom-right to save.
[4] Set up the event

　Since we just need to be able to display a choice, we will
　make it in the simplest way possible.
　Create an event at any position in Sample Map A
　and set a chicken image.
　Open the Command Input Window, change the tab to "2 Choice",
　and enter something (anything will do, such as "Choice")
　in the topmost text entry field. Once entered, press the
　"OK" button at the bottom-right to enter the command.
[5] Verify

　Let's do a Test Play.
　When the choice Event Command launches, the background image
　of the choice should be using a black image. If it doesn't work,
　try reviewing Step 1.
　Great work!
　[Side Note]
· This time we changed the image displayed as the background of the choice when it appears,
but you can also change the image that serves as the background
of the menu opened with the Cancel key during normal play.

This can be changed in User Database Type 16 "Image/Audio Settings",
Item 0 "Window Image File".
Give it a try if you have the time!
