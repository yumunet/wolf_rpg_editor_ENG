【[Battle] I want to configure an enemy group】

[Goal]
　We will create a new enemy group in which 3 "Black Dragons" (a sample game boss) appear,
　then create an event to trigger that enemy group and do a Test Play.

[1]
　Clicking the User Database will
　display a window like the one on the right.
[2]
　Select type "12: Enemy Group".
[3]
　Click on an empty slot
　in the "data" list.

　(Here we will select slot 9.)
[4]
　Enter "Black Dragon x3" as the data ID.

　This is not required for the process to work,
　but let's add it to make things clear.
[5]
　In the fields to the right, enter the following.

　Enemy 1 (Center) → [2] Black Dragon
　Enemy 4 (2 to the left) → [2] Black Dragon
　Enemy 5 (2 to the right) → [2] Black Dragon


　The configuration itself is now done.

　Next, we will create the process
　to make this enemy group appear.
　


　※From here, we will create the event that makes the enemy group appear.
　

【A1】　Here we will create the process to make the enemy appear.

　• First, use the Map Selector to switch to "Sample Map A".

　• Create an event somewhere suitable
 and set it to the chicken image.
　　(Placed at the position shown in the image on the right.)


 → If you are unsure, follow
　  ◆ I want to try creating a new event
　　steps [1] through [5].
【A2】
　Click "■ Show Command Window ■"
　at the bottom right of the Event window.
【A3】
　When the "Insert Event Command" window
　appears, select the tab at the top
　labeled "Other 2".
[4]
　Select "Insert Event" and
　choose Common 24 "◆ Battle Trigger".
[5]
　Next, configure the "Common Event Input (Value)" fields as follows.

• Enemy group to appear: [9: Black Dragon x3]
• Can flee: "0: Can Flee"
• On defeat process: "0: Game Over Event"
• BGM: "3: Combat BGM"

　Once configured, click "Input".
[7]
　Now let's save the map and test the event we created.

　Click "Save (Entire Map)", then
　click "Test Play".
[8]
　Start the Test Play and
　press the Confirm key facing the chicken
　— 3 Black Dragons now appear in combat!


　※ Every time you talk to this event,
　　this combat will trigger again.




[Side Note]　About the order enemies are arranged
　
　Up to 1–7 enemies can appear simultaneously
　in one group. The enemy numbers correspond
　to positions as follows:

　Enemy 6　Enemy 4　Enemy 2　Enemy 1 (Center)　Enemy 3　Enemy 5　Enemy 7

　Generally, entering enemies in order of Enemy 1, 2, 3... works fine,
　but for larger-sized enemies
　it is better to store them in the order Enemy 1, 4, 5.
