【Set Music for a Map】

[Goal]
　Have you gotten the hang of creating RPG maps?
　Now, when you normally play a game, maps usually have music playing, right?
　Music (BGM) is a very important element in games.
　This time, we will show you how to change the BGM of the title screen.
　Here, we will cover how to change to a different BGM registered in the System Database,
　and how to change the BGM by specifying a filename.
　We will also briefly cover how to set BGM when creating a new map.
　By the way, the BGM we will change to is "Staff Roll" (registered as number 1 in the System Database, Piece01_Takumi.mid).

【Steps】

[1]First, open the title map settings
　From the map tree, select the title map.
　Then click the wrench icon to open the map settings screen.
　In the dialog that appears, click on the boxed area.
Right here!
【2A】Changing BGM: From the Database
　It should look like this.
　If the checkbox outlined in blue is unchecked, the screen will appear as shown.
　For this title map, the method of selecting a BGM from the System Database is used.
　From the list in the area outlined in red, select the number of the BGM you want to use.
　In this example, select "2: Staff Roll".
　Note that unless "Enable" is checked in the same area, BGM will not play.
　Furthermore, if "Continue from previous map" is checked, you cannot configure the settings.
Check the red-outlined area!
【2B】Changing BGM: Enter filename
　If you click "Specify filename" shown in the figure, it changes to this view.
　Here, click the "File" button outlined in yellow-green, and select the BGM you want from the dialog that appears.
　This time, select "Piece01_Takumi.mid" from the BGM folder.
　If you want to use a song not registered in the System Database, use this method.
If selected correctly, the address of the selected BGM will be displayed
[3]Let's verify it
　Try launching the game.
　If the BGM you set (in this example, the Staff Roll track) is playing, you're done.
[4]How to set BGM for a new map
　Click the blank page icon to create a new map.
　At this point, a window like this will appear.
　It looks very similar to what appeared in 【2A】, but "Register in DB" is already checked from the start.
　Set the BGM here in the same way as before and the settings will be complete (the method is the same).
　When entering that map, the BGM you set should start playing.
The top says "Create New Map"

[Side Note]

[5]How to set ambient sound (BGS)
　The settings volume is the same as for BGM.
　Change the "BGS to play" section.
Right here! Note that BGS cannot be configured unless "Enable" is checked.
[6]How to register BGM in the System Database
　Please refer to "◆I want to use BGM/sound effect assets in my game".
　BGS is also briefly covered at the end.
[7]For those who become more experienced, this method is also available
　Open the System Database and select type 0.
　The map's BGM is specified in the area shown in the figure.
　You can also change the map's BGM by modifying the value set there.
Right here! Note that when you select "[-3]: Specify by filename", don't forget to enter the filename to use at item number 5.

　That's all for this tutorial. Thank you.
