【I want to set weakness, absorption, and immunity elements on enemies】

　[Goal]

　Elements are settings that determine what a character is good or bad against.
　Through element settings, you can create unique monsters such as a bird weak to lightning,
　a fish immune to ice, a lizard that absorbs fire, and a golem resistant to normal attacks.

　In this topic, we will configure the resistance of the enemy character "Goblin" (already included in the sample game)
　against the "Physical Attack" element to "weakness", "half damage", "immunity", and "absorption",
　and learn how to set the degree of resistance against element attacks.

　【Steps】
　In this topic's 【Steps】, we will first explain under "About Element Resistance" where and what values
　need to be set to create element resistance,
　then in "Element Settings" we will actually configure the Goblin's physical attack resistance,
　and finally in "Confirming the Results" we will show you how to do a Test Play.
　■About Element Resistance
[1] Opening the User Database

　Clicking the User Database will
　display a window like the one on the right.
[2] Selecting the type to edit

　When the window first opens, the "Type" is set to "skill".
　This time the settings will be made in User Database entry 11 "┣ Element Resistance", so
　please click "11:┣ Element Resistance".
[3] Selecting the data to edit

　There is a data list in the middle of the window.
　Please select "31:Goblin" from this list.
[4] Resistance against elements

　Next, configure the effect each element attack has on damage.
　In this article we are setting resistance against "Physical Attack",
　so we will edit "<Physical Attack> Effect [%]" on the right side of the window.
　This is the topmost entry on Page 1.

　If you have not changed the settings since downloading WOLF RPG Editor,
　"[100] 100% [No Change]" should be selected.

　Using this 100% as the baseline, if the resistance setting value is greater than 100,
　the damage increases. (weakness)

　If the resistance setting value is less than 100, the damage also decreases,
　meaning there is resistance. (half damage)

　Setting it to 0 makes that element's attacks completely ineffective. (immunity)

　Entering a value of -1 or less makes the target absorb
　that element's attack and recover HP/SP from it. (absorption)
　■Element Settings
　Here we will introduce how to configure the "Physical Attack" element
　as each of the Goblin's weakness, half damage, immunity, and absorption elements.
　Since all of these are configured in the same location, after setting one,
　please also perform a Test Play (the method is introduced in the next section "Confirming the Results").

　① Setting a weakness element
　Make "Physical Attack" the Goblin's "weakness element".
　Select "[200] 200%" from the dropdown list.
　Once selected, click the "OK" or "Update" button at the bottom right of the window to save.

　② Setting a half damage element
　Configure a "half damage element" so that "Physical Attack" is less effective against the Goblin.
　Select "[50] 50%" from the dropdown list.
　Once selected, click the "OK" or "Update" button at the bottom right of the window to save.

　③ Setting an immunity element
　Make the Goblin's "Physical Attack" an "immunity element".
　Select "[0] 0% [Immunity]" from the dropdown list.
　Once selected, click the "OK" or "Update" button at the bottom right of the window to save.

　④ Setting an absorption element
　Make "Physical Attack" the Goblin's "absorption element".
　Select "[-100] -100% [100% Absorption]" from the dropdown list.
　Once selected, click the "OK" or "Update" button at the bottom right of the window to save.
　■Confirming the Results
　Once you have configured the element resistance, let's do a Test Play.
　If you want to do a Test Play but aren't sure how to create a combat event,
　refer to "I want to trigger a battle"
　to call enemy group entry 31 "Goblin x1" from a map event or similar.

　The image on the right shows the results of a Test Play attacking
　each of the five Goblin variants (normal, weakness, half damage, immunity, absorption)
　with a normal attack (which is a "Physical Attack").
　Even monsters with identical parameters except element resistance can show major differences
　just by changing the element resistance setting.
　That concludes the content of this article. Good work!
