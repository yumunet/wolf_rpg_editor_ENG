[I want to name my variables]

[Goal]
Excluding Self variables and System variables,

assign names to Common Self variables, normal variables,
normal spare variables, and string variables
to make them easier to tell apart.

Version used: [2.x series]

Method 1
Method 2



[1] Method 1
　· Open the Map Selector 
　  and switch to the map where you want to create an event.

　· Create an event somewhere.
　  You don't need to set an image for the event.

　Refer to steps [1]–[5] of ◆An event that activates with the Confirm key.



[2]
　Once created, click
　"■ Show Command Input Window ■"
　at the bottom-right of the Event window.

　1. When the "Insert Event Command" window appears,
　   click and select 3 Variable Operation.

　2. You will see a section labeled "Variable Name Setting".
　   Use this to assign a name to a variable.
　   ※ 5 String Operation also has
　      a similar setting field.

　3. Specify the variable you want to name.
　   For 3 Variable Operation:
　   V n = "Normal variable"
　   V n-j = "Normal spare variable"

　   For 5 String Operation:
　   S n = "String variable"

　   When editing a Common Event:
　   For 3 Variable Operation:
　   CSelf n = "Common Self variable"
　   V n = "Normal variable"

　   For 5 String Operation:
　   CSelf n = "Common Self variable"
　   S n = "String variable"

　   ※ n and j represent numbers.

　4. Type the name you want and click
　   "Assign Name" to the right of the field.

　   The "Copy" button next to step 3 copies
　   the variable name from step 3 into the step 4 field.

　   "Size +1" increases:
　   - For V n: the upper limit of n
　   - For V n-j: the upper limit of j for the specified n
　   - For S n: the upper limit of n

　   You've now assigned a name to a variable.
　   If you want to know Method 2, continue reading.
　   Otherwise, you're done — great work!

[4] Corresponding types:
　 　 　4: String variable names "S n"
　 　　14: Normal variable names "V n"
　15–23: Spare variable names "V n-j"
　※ n and j represent numbers.

　Assign names to the data in the type
　corresponding to the variable names you want to change.

　After making changes, click "OK" or "Update"
　at the bottom-right of the window,
　or press Ctrl+S.
　This saves your changes.

　The changes are now saved and applied.



[5] For Common Self (CSelf) variables

　Refer to step [1] of I want to try creating a new Common Event
　to open the Common Event Editor.

　Click "Self Variable Usage" in the top-right of the window.



[6]

　Did the window shown on the right appear?

　Each entry in that window is a field
　where you can enter a name for a
　Common Self (CSelf) variable.

　After changing entries, don't forget to
　click "Settings Complete" at the bottom-right.

　Clicking Cancel will discard your changes.


　※ About the copy/paste buttons at the bottom-left:
　Copy All
　   Copies the text of all CSelf00–CSelf99
　   from the currently edited Common Event.

　Paste All
　   Overwrites everything with what was copied
　   using Copy All.

　Paste to Empty Fields Only
　   Only fills in empty fields (fields that have
　   at least a space are considered non-empty)
　   with the content from Copy All.



[7] Verify
　Check that the name you set in [5] or [6]
　has been applied correctly.

　From the command list, select
　3 Variable Operation or
　5 String Operation,
　then specify the Common Self variable you changed.
　If the variable now shows a name, you're done!

　Great work!
