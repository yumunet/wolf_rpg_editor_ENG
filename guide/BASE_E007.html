【I want to teach a player character a skill during the game】


[Goal]

• Make it so that examining an event teaches the player character a skill.

Bonus:
• Change the event's dialogue after the skill is learned.

Related topics:
◆ I want to set a player character's skill learn level
◆ I want to add more skill (magic) types


[1]

• First, use the Map Selector to switch to the "map where you want to create the event".

• Create an event somewhere suitable and set it to the chicken image.

For steps, refer to [1]–[5] of ◆ An event launched by the Confirm key.


[2]

Once created, click "■ Show Command Input Window ■" at the bottom right of the Event window.

1. When the "Insert Event Command" window appears,
　click the Other 2 tab. (Image 1)

2. Confirm that "Insert Event" (circled in the image) is selected. (Image 2)
　If the circle next to the text has a dot in it, it is selected.
　※ If it is not selected, click it to select it.

3. Clicking the area circled by box 3 will display a list
　of available Common Events.
　Since this is a skill-learning event, (Image 3)
　select "Common 8: ○ Special Skill Increase/Decrease".


[3]

Next, edit the "Common Event Input (Value)" fields.

1. Target Player Character: Select the character who will learn the skill.

2. Which Skill?: Select the skill to be learned.

3. What to do?: Choose whether to learn or forget the skill.
Since we are learning here, select "0: Learn".

4. Finally, click the Input button at the bottom right.

The skill-learning process is now complete.


[4]


The skill-learning event is complete up to this point.
However, as-is the player will learn the same skill every time they examine the event.
Since you cannot actually learn the same skill more than once, this is fine as-is,
but let's also make the event display different dialogue after the skill has been learned.
※ Click here if you are not interested in this part.

For more details on the following process, refer to [5] onwards in ◆ A treasure chest that stays open once opened.


[5]


• Find and click "Variable Operation" in the "Insert Event Command" window.

• Confirm the parts highlighted with red lines. Configure them to match the image on the right.
　
• Finally press Input at the bottom right.

What was configured here is a process that assigns "1+0=1" to "Self Variable 0 of this event".


[6]


The image shows the top-left of the Event window.
Pay attention to the part enclosed in yellow.

1. Click the "New Page" button.

2. Click the "Page 2" button to open Page 2.


[7]


Page 2 is the state of the event after the skill has been learned.
As-is it would become a transparent event, so set the graphic to the same "chicken" as Page 1.

1. Double-click the gray square.

2. Select Chicken.png from [CharaChip].

Once done, click the "OK" button.


[8]


Let's configure the "Event Trigger Condition".
Pay attention to the part enclosed in purple.
　
1. Click the leftmost checkbox to add a checkmark.

2. Click the "Self" button to select it.

3. Select "Self Variable 0".

4. Enter the value "1".

5. Select "Same as".

This means Page 2 will only launch when Self Variable 0 of this event equals 1.


[9]


Now let's enter the dialogue.

Click "■ Show Command Input Window ■".

1. Click the Show Message tab.

2. Enter the dialogue you want to display in the text input field.

3. Finally, click the Input button.


【Save】


Save the map and do a Test Play!
Clicking the Test Play button in the Event window will save automatically.


【Test Play 1】


Unlike items, no message is displayed when a skill is learned.
(If you want a message to appear, refer to [9] and add a message to Page 1.)
Press the Cancel key to open the menu and confirm from Special Skills.

※ If something goes wrong...
Check the settings in [3].


【Test Play 2】


Now that we have confirmed the skill was learned, let's also check whether the dialogue has changed.

Examine the event again.
If the text you entered is displayed, you have succeeded.



This concludes "I want to teach a player character a skill during the game".
Great work!
