【I want to create a shop event】

[Goal]
　In "Sample Map A" of the sample game,
we will create a shop event where you can buy a "Medicine Bottle".

[1]
　• First, use the Map Selector to switch to "Sample Map A".

　• Create an event somewhere suitable and
set it to the chicken image.
(Placed at the position shown in the image on the right.)


　→ If you are unsure, follow
　　　◆ I want to try creating a new event
steps [1] through [5].
[2]
　Click "■ Show Command Window ■"
　at the bottom right of the Event window.
[3]
　When the "Insert Event Command" window
appears, select the tab at the top
labeled "Other 2".
[4]
　　• Select "Insert Event" and
　　choose "Common 28: [1] Shop Initialize".

　　• Once selected, click "Input".
[5]
　　Select "Insert Event" again and
　　choose "Common 29: [2] Shop Add Product".
[6]
　　Next, configure the "Common Event Input (Value)" fields as follows.

　　• What to add?　"1: Item [A]"
　　• [A] Item　"0: Medicine Bottle"
　　• [B] Weapon　"-1:"
　　• [C] Armor　"-1:"

　　• Once configured, click "Input".
[7]
　　Select "Insert Event" once more and
　　choose "Common 30: [3] Run Shop Process".
[8]
　　Next, configure the "Common Event Input (Value)" fields as follows.

　　• Shop type　"3: Both Buy & Sell"
　　※ The other options are [1: Sell only / 2: Buy only],
　　but here we set it to allow
　　both buying and selling.

　　• Once configured, click "Input".
[9]
　　The event is now complete.

　　• Please confirm that the Event Command display on the right shows:
　　■ Insert Event: Common 28 [[1] Shop Initialize]
　　■ Insert Event: Common 29 [[2] Shop Add Product] / 1: Item [A] / 0: Medicine Bottle / -1 / -1
　　■ Insert Event: Common 30 [[3] Run Shop Process] / 3: Both Buy & Sell
　　as displayed.
　　If it looks like this image, you are good to go.
　　※ Note that if the order of the events is wrong, the shop will not run correctly.
[10]
　Now let's save the map and test the event.

　Click "Save (Entire Map)", then
　click "Test Play".
[11]
　　Try approaching the event you created and pressing the Confirm key.

　　If the shop event runs as shown in the image on the right, you have succeeded.

　　Great work! You're done.


[Side Note] What if you want to add weapons or armor as products?

　　• In step [6]'s "Common Event Input (Value)" settings,
　　select "2: Weapon [B]" or "3: Armor [C]" for "What to add?",
　　then configure the product under "[B] Weapon" or "[C] Armor".



　　What if you want to add 2 or more products?

　　• Repeat steps [5] and [6] as many times
　　as needed to add products.

　　※ Again, pay attention to the order of events.
　　If it looks like the image on the right, you are good to go.
