[I want to add more item (armor) types]

[Goal]
　We will add a new armor item.
　Here, we will create one new piece of armor.


[1]
　Click the User Database button to open
　a window like the one shown on the right.
　　
[2]
　When the window first opens,
　the "Type" will be set to "skill".

　Please click "4:armor"
 →
[3]
　Click on an empty slot
　in the "Data" list.
　
[4]
　Fill in the fields on "Page 1".

　The following fields are
　filled in here:

　- Armor name
　- Description
　- Equipment type

　- Use item to activate special skill?
　- Special skill to activate

　- Max HP change value
　　(HP increases by 10)
　- Defense power modifier
　　(Defense +10 when equipped)
　- Evasion rate modifier
　　(Evasion rate +20% when equipped)

　- Sale price
　　(0 = cannot be sold)

　* For reference values, see the
　　"Blue Clothes" armor in the sample game.

　* For fields you are unsure about,
　　try copying the sample game armor.
→
[5]
　Fill in the fields on "Page 2".

　In this example, nothing
　has been changed here.

[6]
　Press the Update button or the OK button.
　The new armor has been saved.
