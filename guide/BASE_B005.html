【I want to trigger an event during combat】

[Goal]
　Trigger events at the following timings:
　Immediately after combat starts, between all enemies being defeated and combat ending
　At the start of a turn, at the end of a turn 

[1]
　The WOLF RPG Editor version used here is [2.x series].
　First, let's open the Common Event list.
　Left-click  in the circle on the right.


[2]
　Once the Common Event editor opens,
　search the Common Event list on the left side of the window for
　203: ○[Modifiable] Combat Start Process.
　It is quite far down, so you can drag the scrollbar all the way down at once.
　The 4 Common Events we will use are 203 through 206.
















[3]
　Let's display a message when combat starts.

　Select 203: ○[Modifiable] Combat Start Process
　from the Common Event list.

　Select Show Message from the Event Commands.

　Enter something appropriate for the message content, like:
　Combat Start Process

　Don't forget to save by pressing Ctrl+S
　or the Update button.


















[4]
　Now let's trigger a combat and check what happened.

　Refer to ◆ I want to trigger a battle
　to trigger a combat against a suitable opponent.

　Did the message display after all enemies appeared?

　If it did, you succeeded.

　>>> Possible reasons if the message does not display: <<<
　1. You modified a Common Event other than 203: ○[Modifiable] Combat Start Process.
　2. 203: ○[Modifiable] Combat Start Process is not being loaded for some reason.
　　　>> Open Common 188: X◆ Combat Process
　　　>> Confirm that ○[Modifiable] Combat Start Process is being called on line 43.

















[5]
　Similarly, try setting up [3] for:
　204: ○[Modifiable] 1-Turn Start Process  <188: X◆ Combat Process, line 71>
　205: ○[Modifiable] 1-Turn End Process  <188: X◆ Combat Process, line 118>
　206: ○[Modifiable] Post-Combat End Process  <188: X◆ Combat Process, line 181>
　and confirm carefully in a Test Play at what timing each one is called.

　You can now trigger events at certain fixed timings during combat.

　Good work!
