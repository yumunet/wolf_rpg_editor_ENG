【◆ I Want to Easily Create High-Quality Walking Character Assets】

[Goal]
Using the "Character Graphic Composer"
included with the WOLF RPG Editor full set,
you can easily create high-quality walking character assets.
As one example of how to use it, we will create
the walking character asset for the sample game's
player character "Wolfar" (the person in blue clothing).

However, there is one note when using the composer.
The parts included in the composer were made with considerable effort by asset creators,
so please do not upload assets made with the composer
to asset submission threads or similar places.

[1] Open the "Data Collection" folder
inside the "WOLF_RPG_Editor" folder.
[2] Extract and open
"Character Graphic Composer_Please Extract Before Use".
[3] Double-click "CharaGraMaker" inside
the "graphic composer" folder and
a window like the one on the right will open.
The "Exit" button in the bottom-right of the opened window
will close the composer without warning.
Please be careful.
[4] Now let's choose some parts and
create a graphic.
From "Head", select "Long Hair [Silver].png".
A female face with silver-colored long hair
should appear on the right.
　　(The image shows only 4 columns wide,
　　but the actual graphic has 6 columns and 4 rows)
[5] Next, from "Eyes", select "Eyes [Blue1].png".
Since the original eye color is already blue,
you may not feel much of a change.
Feel free to try other colors as well.
[6] Next, from "Skin", select "[Female] Base.png".
Since this part determines the skin color of the whole body,
selecting it will display the entire character.
[7] Next, let's put some clothes on. From "Clothes",
select "[Female] Snow Country Outfit [Blue].png".
You can gradually start to recognize a familiar figure.
By the way, the gender labels like [Female] or [Male] only mean
that the shadows line up correctly when paired with a "Skin" of the same gender —
the visual gender of the character is primarily determined by the "Head" part.
[8] Finally, selecting "Feathered Hat.png" from "Accessories"
places a blue hat on top of the character's head.
The sample game character
"Wolfar" is now complete.
[9] Now that the walking character asset composition is done,
let's save the completed graphic.
Click "Save File" near the bottom-right of the graphic area
and save it to the "CharaChip" folder.
(Those who want to read the extra notes, please continue without closing.)
This concludes the creation of a walking character asset
using the character graphic composer. Thank you for your hard work.
　
【Extra Note 1】 "Color"
A description of the "Color" feature
of the character graphic composer. First, look at the image on the right.

Wolfar's outfit has been changed from blue to green,
and the hair from silver to blonde.
However, no parts were changed from the ones
used in the description above.
Instead of changing the parts themselves,
the "Color" of the parts has been changed to create a color variation.

"Color" refers to the hue, and changing it
shifts the color through a rainbow-like sequence.
This allows you to change the hue of a part.
However, a rule of hue is that pure gray tones (white to black)
will not change color even when the hue is adjusted.
The leftmost position is the default — blue (the default color).
Try dragging the "Color" slider for "Clothes"
and watch the outfit color change to green, orange, purple, and more.
In this sample, the colors of 3 parts — Head, Clothes, and Accessories —
have been changed, so please give it a try.
If mouse control feels a bit tricky,
use the left and right keyboard keys for fine adjustment.
【Extra Note 2】 "Saturation"
For those continuing to read, please reset
the hue you changed earlier back to the default (leftmost position of the bar).
This time, we will create a graphic like this.
The hair has been changed from silver to blue, and the outfit from blue to gray.
Again, no parts were changed — instead, "Saturation" was adjusted.
"Saturation" refers to the vividness of a color.
Higher saturation makes the color more vivid; lower saturation makes it gray.
For "Skin" saturation, it's generally better not to change it
unless you're making something like a stone statue.
In the image, the saturation of Head, Clothes, and Accessories has been changed.
Note that the default position for saturation is the center, and once changed it can be hard to undo,
so there is a reset button for saturation.
Pressing it restores the saturation of all parts to their defaults.
【Extra Note 3】 "Random Composition"
This is the last extra note. We've been using Wolfar's graphic
for both "Color" and "Saturation",
but we won't be using it here, so first press "Exit" to
close the window, then open it again.
Random Composition is a very convenient feature
that automatically selects parts and
generates a character graphic for you.
How to use: simply check the "Random" checkbox
for each part category you want to randomize.
Try checking all 3: "Head", "Eyes", and "Clothes".
Select "[Female] Base.png" for "Skin"
and leave its Random checkbox unchecked.
Click "Random Composition" at the bottom-right,
and it will randomly select one part each from
"Head", "Eyes", and "Clothes"
to create a character graphic.
You can redo it as many times as you like until you find something you love.
If you get something you like, you can save it as-is,
or you can add your own touches to it.
If you check "Also Randomize Color" and
do a Random Composition, it will also randomize the hue
while compositing the graphic.
