[The initial player character image in Game Basic Settings isn't showing]

　When using the Basic System, setting the "Initial Player Character Image" in "Game Basic Settings" will have no effect.
　This is because immediately after the game starts, the Basic System performs the following process:
　"Automatically changes the character graphic to the initial party set in Changeable DB Type 3."
　As a result, the character displays the graphic from that "initial party" setting, not the "initial player character image".

　So to set a player character's graphic, please see
　◆ I want to set the player character's walking graphic.

　→ If you're unsure how to configure the initial party, refer to ◆ I want to configure the starting party.
