[I want to try creating a new event]

[Goal]
　We will create a new event inside "Sample Map A" in the sample game:
　a chicken event that says "Cock-a-doodle-doo♪" when you talk to it.


　※What is an event?
　　For now, think of it as "an NPC in an RPG village that says something when you talk to it" — that's good enough.

[1]　
　Open the Map Selector and
　select "1: Sample Map A".
　
[2]
　Switch to the Event Layer ,
　then right-click just above the game start position.
　A menu like the one shown will appear.

　Select "Create/Edit Event".
[3]　
　The Event Settings window
　shown on the right will appear.

　
[4]
　First, set the event's walking graphic.
　Double-click the area outlined in red.
[5]　
　A window like the one on the right will appear.
　Select "Chicken.png" and press "OK".


　※You can click the character image
　　on the right side to change
　　the facing direction of the event.
[6]　
　The chicken graphic has been
　set successfully.
[7]
　Next, add the process that makes
　the chicken speak.

　Click the
　"■ Show Command Input Window ■"
　button at the bottom-right of the Event window.
[8]　
　The "Insert Event Command" window will open.
　Make sure the selected tab is "Show Message",
　as shown on the right.
[9]　
　Type "Cock-a-doodle-doo♪" in the text input field,
　then click the "Input" button.
[10]
　If everything looks like the image on the right,
　you've succeeded.

　The Trigger Condition is already set to
　"Execute on Confirm key",
　so this event will work as a
　talk-triggered event right away.
[11]　
　Click the "Save (Entire Map)" button
　at the bottom-right of the Event window,
　then click "Test Play"
　to save this map (with the new event)
　and start a test play.
[12]　
　Select "New Game" on the title screen,
　skip through the intro, then
　press the Confirm key while facing the chicken
　to talk to it!

　It said "Cock-a-doodle-doo♪"!

　Your first event is complete!
　Great work!




　　

[Side Note]
　Can I put anything I want in the event "Name" field?

　When you set a graphic, the filename is automatically entered
　in the "Name" field, but you can change it
　to anything — "Chicken", "Cock-a-doodle-doo Event", whatever you like.

　So what is the "Name" actually used for?
　While editing the Event Layer, hovering the cursor over an event
　will show that event's "Name" in the title bar.
　
　That's all it does — so a cool name won't get you anything!
