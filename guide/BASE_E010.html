【I want to create random encounters】

[Goal]
We will create a "random encounter" — where walking in a certain area causes
you to run into monsters — in the building at the bottom-right of Sample Map A.

[1] While watching the coordinates (○,□) at the top of the WOLF RPG Editor window,
find coordinate (33,20) and double-click it.
The Map Event Editor will open.
[2] Click "■ Show Command Input Window ■".
[3] Click "Other 2".
[4] Select "Common 32: ◇ Random Encounter Process".
[5] Enter "10" in the "Steps Before Encounter" field, then
click "Input" at the bottom right of the window.
This creates an event where you encounter a monster approximately every 10 steps.
[6] Next, configure the range of the random encounter and its Trigger Condition.
Set "Contact Range Extension" to "X:10, Y:4" to match the building's floor, and
set "Trigger Condition" to "Player Touch".
[7] That's all the configuration. Please do a Test Play.
You should start encountering monsters approximately every 10 steps.
If you never encounter at all, there are mainly four possible causes:

◇ [5] "Steps Before Encounter" is wrong.
┗ If it is still the default value of "100", you won't encounter until you walk about 100 steps.
◇ [5] You didn't press "Input" at the bottom right of the window.
┗ Even if you configure the content of the event to call, it won't be saved unless you press "Input".
◇ [6] "Contact Range Extension" is wrong.
┗ If it is still the default value of "0,0", step counting and encounters will only occur on that one tile.
◇ [6] "Trigger Condition" is wrong.
┗ The condition for launching by walking is "Player Touch".
　 With "Execute on Confirm key", you would need to mash both the Confirm key and the movement key simultaneously to encounter.

That's how to create random encounters. Great work!
