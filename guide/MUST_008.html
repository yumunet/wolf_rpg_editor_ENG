[I want to change the title image [Sample game only]]

[Goal]
　Change the title image of the WOLF RPG Editor sample game to a different image.


Before		  → .
　
After
[1]　
　Open the "Data" folder
　inside the "WOLF_RPG_Editor" folder.
　
[2]
　Open the "SystemGraphic" folder.
　
[3]　
　There is a file called "TitleGraphic".
　This is the title image shown
　in the sample game.
　
　The file format is PNG,
　and its size is 320×240 pixels.
　
　
[4]　
　Prepare an image file with the same size
　(320×240 pixels) and format (PNG) as [3].
　
　
[5]
　Overwrite "TitleGraphic"
　with the image file you prepared in [4].
　
　
 →
[6]　
　Run Test Play and
　the title screen has changed to your image!
