【I want to add or remove party members】


[Goal]
We will create an event where
a new member joins the party when you talk to them.

[1]
• First, use the Map Selector to switch to the "map where you want to create the event".

• Create an event somewhere suitable and set it to the chicken image.

For steps, refer to [1]–[5] of ◆ An event launched by the Confirm key.

[2]
Once created, click "■ Show Command Input Window ■" at the bottom right of the Event window.

When the "Insert Event Command" window appears, click the Other 2 tab. (Image ①)

[3]

[2] Image ② Click the ▼ button in the Insert Event section.
A list of Common Event names will appear — select "Common 5: ○ Member Increase/Decrease".

Next, look at the Common Event Input (Value) section. [2] Image ③
For "Process", select "0: Add to Party". For "Who?", select the member to add.
In this example, we are adding Yuichi to the party, so we selected "2: Yuichi [Var 0]".

Once configured, [2] Image ④, click the Input button.
The event is now complete.

【Bonus】
Since that alone feels a bit bare, let's also enter some dialogue for when the member joins.

Once you're done entering, don't forget to save!

【Test Play】
Finally, do a Test Play to confirm it works.

※ If something goes wrong...
Check whether "Process" in image ③ of [3] is set to "1: Remove from Party" or
"2: Update Walking Image Only" by mistake.
Also check whether you selected the wrong character under "Who?".

This concludes "I want to add or remove party members".
Great work!





[Side Note]
If you want to remove a character who is already in the party,
select "1: Remove from Party" for "Process" in image ③ of [3].

Then, just like when adding a member, select the target character under "Who?".
