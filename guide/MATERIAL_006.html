【Use a Preferred Font in Your Game】

[Goal]
In WOLF RPG Editor, you can change the font used in various places
such as message displays and menus.

Changing the base font dramatically changes the atmosphere of your game.
For example, the MS Gothic font registered as the default in WOLF RPG Editor
is a crisp font that works in any context, but
a Mincho-style font gives a delicate, Eastern impression.
For works with a Japanese or Chinese-inspired Asian aesthetic,
Mincho might be preferable.
All of this becomes possible once you register the font for use
in WOLF RPG Editor.

This time, we will learn how to change the
base font "MS Gothic" used in WOLF RPG Editor to "MS Mincho",
and how to register fonts.

Note that the technique introduced in this article is
"registering a font installed on your computer for use in WOLF RPG Editor" —
for "using an already registered font in practice", see
the article "Change the Font in Messages".

★Note!
　When using assets, always read and comply with the asset's terms of use.
Also see "What are the rules for using assets?".
　Font-related terms of use are particularly strict. See the side note for details.
【Steps】

1. Register the base font

　In "Game Settings", select "Open Game Basic Settings"
　to open the "Game Basic Settings" window.
　You can configure this in the item
　"Base Font (blank = MS Gothic)".
　Enter "MS Mincho" in the "Base" input field
　and press the "OK" button at the bottom right of the window.
　Enter "MS" in full-width characters, and the space after it in half-width.
　Getting this wrong will likely cause it not to be recognized.
Right here.
Please make sure to enter accurately!
2. Verify

- Try Test Play. You should be able to tell that the font has already changed
　from the title screen choices.
　The font in talking events and the menu screen should also
　have changed to MS Mincho.

- If "It hasn't changed!", review step 1.
　Full-width and half-width characters are distinguished.
　With this, the configuration is done. Great work!

- If you want to return to the original font, follow step 1 again
　and enter "MS Gothic" in the "Base" input field,
　then press OK to restore it.
　Again, enter "MS" in full-width characters, with a half-width space after it.
The first scene of the sample game.
Does the atmosphere feel different?

[Side Note]

1. How to change fonts in specific parts only

This time, we changed the base font from "MS Gothic" to
"MS Mincho", so everywhere MS Gothic was used
is now MS Mincho.
For those who want to change only specific text sections
or "change only part of the message display",
the article "Change the Font in Messages"
explains the technique for using a desired font wherever you want,
so please refer to that.
2. How to use fonts not installed on your computer

- Fonts freely released or sold on the web can be used
　if the author permits their use. However, font usage is
　actually quite complex —
　terms of use can be difficult, some may prevent you from distributing your game,
　or a file that should work somehow doesn't,
　or a font you installed doesn't display correctly on another computer.
　Fonts are among the most difficult assets to handle.

- First and foremost, check the terms of use carefully.
　Particularly tricky is "prohibition of redistribution" — in WOLF RPG Editor,
　there is no way to include a font while also
　avoiding redistribution of that font.
　If a font completely prohibits redistribution,
　you'll need to give up on using it in WOLF RPG Editor.

- Place the font file in the same location as Game.exe,
　or in the Data folder.
　Then, follow step 1 and specify the font name correctly
　to install the font.
