【I want to change a player character's stats during the game】


[Goal]

• Create an event where talking to an NPC raises the player character's stats.


[1]

• First, use the Map Selector to switch to the "map where you want to create the event".

• Create an event somewhere suitable and set it to the chicken image.

For steps, refer to [1]–[5] of ◆ An event launched by the Confirm key.


[2]

Once created, click "■ Show Command Input Window ■" at the bottom right of the Event window.

1. When the "Insert Event Command" window appears,
　click the Other 2 tab. (Image 1)

2. Confirm that "Insert Event" (circled in the image) is selected. (Image 2)
　If the circle next to the text has a dot in it, it is selected.
　※ If it is not selected, click it to select it.

3. Clicking the area circled by box 3 will display a list
　of available Common Events. (Image 3)
　Since this is a stat increase/decrease event,
　select "Common 7: ○ Stat Increase/Decrease".


[3]

Next, edit the "Common Event Input (Value)" fields.

1. Target Player Character: Select the character whose stat you want to increase or decrease.

2. Which Stat: Select the stat you want to increase or decrease.
　Selecting "Growth Rate" lets you set the amount gained per level-up.

3. By How Many Points: Set the value to increase or decrease (or assign).
　Entering a negative value here will decrease the stat.

4. Increase/Decrease or Assign: Choose whether to increase/decrease from the current value,
　or to overwrite it with the value entered in 3. By How Many Points.

5. Finally, click the Input button at the bottom right.

The stat increase/decrease process is now complete!


[4] Test Play

In this example:
Target Player Character: "Wolfar"
Which Stat: "Max HP"
By How Many Points: "100"
Increase/Decrease or Assign: "Increase/Decrease" — to raise the stat.
If the stat has changed as shown, you have succeeded.

※ If something goes wrong...
Check whether the input fields and target were set correctly when configuring in [3].
