【I want to switch the character assets used between 4-direction and 8-direction】

　[Goal]

　Many of the assets bundled with WOLF RPG Editor's Basic System
　are 8-direction assets that support diagonal facing, but many assets available
　from other sources only support 4 directions.
　The game cannot automatically identify and configure these separately,
　so you need to select the correct type manually.
　This time we will learn how to configure these settings.

　【Steps】
[1] Open the Settings screen

　Select "Open Game Basic Settings" from "Game Settings",
　and open the "Game Basic Settings" window.
[2] Change the setting

　You can configure this under the item "Character Image Direction Type".
　If the image asset supports diagonal facing, select "8-Direction Support";
　otherwise select "4-Direction Support",
　then press the "OK" button at the bottom-right of the window.
　The settings are now done. Great work!
　[Side Note]
　■ If the character display looks wrong

· After changing the settings in the steps above, you may have panicked during Test Play
　if the character was
　"only showing half of the character!" or conversely
　"the character split into two people!".
　Please be reassured that this can be fixed without any downside.
　The cause is either using an 8-direction asset with 4-direction settings,
　or conversely using a 4-direction asset with 8-direction settings —
　in other words, you changed the previously correct settings to the wrong ones.
　Simply correcting it to the right settings as described in the steps above will restore it.
· If you use an 8-direction image with 4-direction settings,
　or conversely configure a 4-direction image with 8-direction settings,
　as also covered in the article "Character splits in half / character splits into 2",
　issues will occur such as the character splitting into two people or only the top half being displayed.
　The key point is "whether the character image supports diagonal facing" —
　please verify the asset type carefully.
　■ The difference between 8-direction assets and 4-direction assets

· An 8-direction asset is an image with characters arranged in 6 columns × 4 rows, capable of facing diagonally.
　Most of the character chip assets in the CharaChip folder, and
　the character chips that can be created with the graphic composer bundled (compressed) in
　WOLF RPG Editor's Basic System data collection are of this type.
· A 4-direction asset is an image with characters arranged in 3 columns × 4 rows, incapable of facing diagonally.
　The image "[Animal]Halcyon_ipu" in the CharaChip folder is
　one of these, and most images available from other asset sites are also
　4-direction assets.
· As shown, 8-direction assets and 4-direction assets are similar but fundamentally different, and
　as of Ver2.02, they cannot be mixed and used together by any means.
　Modifying 8-direction assets to force their use is,
　of course, a rule violation if the asset's creator has not permitted image modification,
　so please be careful.
