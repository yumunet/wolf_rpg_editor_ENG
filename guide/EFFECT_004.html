【I want to change the text color in a message】


WOLF RPG Editor has a feature called special characters.
Let's use one of them to change text color.

[Goal]

　1. Configure text colors in the System Database.
　
　2. Use special characters to display text in the configured color.


【Steps】

―― [1] Text Color Settings ――

　First, press to open the System Database.
　
　From the upper-left list, select "Type 12: Text Color".
　Avoid entries that already have something, and choose an empty data slot.
　This time, as an example, let's enter something in slot 4.
　
　Give it an easy-to-understand name and enter the values.
　The larger the value, the more intense that color appears.
　Click OK when finished.
　
　※Like light, when all three colors (red, green, blue) are mixed together the result is white,
　 　and as values approach 0 the result gets blacker.
　 　Gray can be reproduced by setting all three color values approximately equal.
　 　Conversely, to make a color vivid, increase the value of that color
　 　and also increase the difference from the other colors.
　

―― [2] Create an Event ――　

　Next, use the Map Selector to switch to "Sample Map A".
　Create an event somewhere easy to find.
　Use chicken for the graphic.
　(Reference: 【Event launched by Confirm key】)
　

―― [3] Create the Message ――　

　Now enter the message text.
　
　Click "■ Show Command Input Window ■" in the lower right of the Event window
　to open the Event Command Input window.
　
　Select "1 Show Message" from the left list.
　In the input box, enter the following text:
　
　"\c[4]WOLF RPG Editor"
　
　The string \c[(number)] is the special character that changes text color.
　Since we configured text color slot 4 this time, the number inside [ ] is 4.
　

　The event is complete.
　Save the map and do a Test Play.
　
　Try talking to the event.
　If the text color has changed, it was a success.
　Good work!
　

[Side Note]

　Text color changes using special characters are valid only within that message.
　
　To change the default text color,
　modify the settings for [0: Default Color] at the top of
　System Database [12: Text Color].
　
　You can also change the text color multiple times within a single message.
　To revert to the default color in the middle, enter \c[0].
