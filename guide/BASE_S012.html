[I want to change basic terms used in the Basic System (e.g. "Gold" to "Yen")]

[Goal]
　We will change the currency unit from "Gold" to "Yen".


"Gold"		  →
"Yen"
[1]
　Click the User Database button.
　
　A window like the one shown
　on the right will appear.
　
[2]
　When the window first opens,
　the Type is set to "skill".
　
　Scroll the scrollbar down and
　select "15: Base System Settings".
　
 →
[3]
　Select "Page 2" in the upper right.
　
[4]
　Change "[Status] Currency Unit"
　from "Gold" to "Yen".
　
 →
[5]
　Run a Test Play and
　"Gold" has changed to "Yen"!
[Side Note]
　Other terms can also be changed
　using the same method.

　In the image on the right,
　"SP" was changed to "MP" on "Page 1".
