[I want to add more item (weapon) types]

[Goal]
　We will add a new weapon item.
　Here, we will create a Poison Dagger (enhanced dagger with a chance to inflict poison).


[1]
　Click the User Database button to open
　a window like the one shown on the right.
　　
[2]
　When the window first opens,
　the "Type" will be set to "skill".

　Please click "3:weapon"
 →
[3]
　Click on an empty slot
　in the "Data" list.
　
[4]
　Fill in the fields on "Page 1".

　The following fields are
　filled in here:

　- Weapon name
　- Description
　- Equipment type

　- Use item to activate special skill?
　- Special skill to activate
　　(Guard effect when used)

　- Attack animation
　- Attack power modifier
　　(Attack +50 when equipped)

　- Sale price

　* For reference values, see the
　　"Dagger" weapon in the sample game.

　* For fields you are unsure about,
　　try copying the sample game weapons.
→
[5]
　Fill in the fields on "Page 2".

　- Status condition to inflict
　- Infliction chance
　　(50% chance to inflict poison)
→

[6]
　Press the Update button or the OK button.
　The new weapon "Poison Dagger" has been saved.




[Side Note] Test Play

　When you equip the Poison Dagger and enter combat,
　you can inflict poison on enemies as shown
　in the image on the right.
　(The poison infliction rate is 50%.)
