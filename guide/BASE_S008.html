[I want to add more item (consumable) types]

[Goal]
　We will add a new consumable item.
　Here, we will create the "WOLF RPG Editor Official Guide" item (effect: fully heals all allies).


[1]
　Click the User Database button to open
　a window like the one shown on the right.
　　
[2]
　When the window first opens,
　the "Type" will be set to "skill".

　Please click "2:item"
 →
[3]
　Click on an empty slot
　in the "Data" list.
　
[4]
　Fill in the fields on "Page 1".

　The following fields are
　filled in here:

　- Item name
　- Description text
　- Effect type

　- Effect target
　- HP recovery
　　(recover 100% of max HP)
　- SP recovery
　　(recover 100% of max SP)

　- Status conditions healed
　- Message after use
　- Sale price
　　(500 gold)

　* For reference values, see the
　　"Medicine Bottle" and
　　"Heal Scroll" items in the sample game.

　* For fields you are unsure about,
　　try copying the sample game items.
→

[5]
　Press the Update button or the OK button.
　The new item "WOLF RPG Editor Official Guide" has been saved.
