【◆How to Load a Basic System Bug Fix Version】

[Goal]
The Basic System is updated with bug fixes by WOLF RPG Editor's creator SmokingWOLF,
based on reports received through the official site's "bug report thread" and similar channels.
When there is a major version update or similar event,
bugs that existed in your current Basic System may have been fixed.
This topic explains how to re-load the Basic System and introduce the latest version.

[1] Preparation before installation
From the official site's "WOLF RPG Editor Download"
page, download the "Basic System Bug Fix Version".
(If necessary, also download the "Database Update")

Also, this process involves making significant changes to the system.
When making large changes like this,
taking a backup is important.
①Back up the entire game data
Copy the game being developed as a whole folder,
or otherwise create a backup.
②Back up Common Events
We will now load the bug fix version of the Basic System,
but Common Events 0–80 will all be overwritten.
If you have made any modifications, save those parts.

As an example, save Common Event 036,
○[Modifiable] Combat Start Process.

From the Common Event editor,
select event 036 ○[Modifiable] Combat Start Process.

Click "Save (Single)" at the bottom left.
Save to a backup location (somewhere easy for you to find)
as a .common file.

※If the modified Common Events are consecutive,
using "Save Multiple" is a good option.
[2] About the contents of the downloaded files
Check the contents of the downloaded files.

※The following description uses data from the "WOLF RPG Editor Download" page
as of February 13, 2011.

"Basic System Bug Fix Version"
- Common000_CallBasicSystemVer144.common

"Database Update"
- SystemDB23_SparseVariable9_data0_Call[UpdatedInVer1.41].dbtype
- UserDB015_BaseSystemSettings[UpdatedInVer1.40].dbtypeset
[3] Update the "Basic System Bug Fix Version"
For standard updates, this alone is sufficient.

①From the Common Event editor,
select Common Event 000 (○Item Increase/Decrease).
②Click "Load" at the bottom left.
③Open "Common000_CallBasicSystemVer144.common".
④A message will display saying "This is multiple Common Event data. Overwrite Common Events 0 to 80?" — select "OK".
To verify the update was applied correctly,
check the version and contents of
Common Event 080 【Basic System VerX.XX Description】.
⑤After confirming the Basic System was correctly updated,
click "Update" at the bottom right.
※If this operation is not performed, the update will not take effect, so be careful.

That's all for the update.
[4] Update the "Database Update"
When there is a major update to WOLF RPG Editor,
this also needs to be updated.
This time we will update the "System Database" and
"User Database".
【Updating the System Database】
①From the System Database editor,
select System DB type 23 (Spare Variable 9),
data 0 (◆◆[Judgment Result]◆◆).
②Click "Load" at the bottom left.
③Open "SystemDB23_SparseVariable9_data0_Call[UpdatedInVer1.41].dbtype".
④A message will display saying "This dbtype is a type settings & full data file.
Replace the entire type including all data?" — select "OK".
⑤After confirming that System DB type 23 was correctly updated,
click "Update" at the bottom right.
※If this operation is not performed, the update will not take effect, so be careful.
【Updating the User Database】
①From the User Database editor,
select User DB type 15 (Base System Settings),
data 0 (System).
②Click "Load" at the bottom left.
③Open "UserDB015_BaseSystemSettings[UpdatedInVer1.40].dbtypeset".
④A message will display saying "This dbtypeset is a type settings file.
Replace the type settings?" — select "OK".
⑤After confirming that User DB type 15 was correctly updated,
click "Update" at the bottom right.
※If this operation is not performed, the update will not take effect, so be careful.
That's all. Great work!
