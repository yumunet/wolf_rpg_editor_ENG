【Event launched when the opponent touches you】

[Goal]
　Create an event that launches when the opponent touches you.

　Here, inside "Sample Map A" in the sample game, create a new event for a chicken that approaches Wolfar
　and displays the message "Yuichi attacked!" when it makes contact.

　

[1]
　・First, use the Map Selector to switch to "Sample Map A".
　
　・Create a new event at a suitable location.

　→ If you are unsure, refer to steps [1]～[3] of ◆ I want to create a new event.
[2]
　Let's set the event's graphic.
　
　Double-click the area enclosed in a red box.
[3]
　A window like the one on the right will appear.
　
　・From the list on the left side of the window,
　　select Chicken.png.

　・Various facing directions of the chicken
　　(character chip) will then display on the right.
　　Selecting a chicken facing a different direction will change
　　the direction the chicken faces,
　　but leave it as-is for now without changing it.
　
[4]
　The chicken has been set as the event graphic.
　Next, change the
　"Trigger Condition" setting to "Event Touch" so the event launches on contact.

　This makes the current event
　"launch when the event makes contact with the player character".


　Also, set the following 2 additional settings at the same time:
　・Change "Movement Route" to "Approach Player"
　　so the event approaches the player character.
　・Change "Move Frequency" to "0: Every Frame"
　　so the event does not stop after each step it takes.
[5]
　Finally, create the event contents.
　
　Click "■ Show Command Window ■"
　at the bottom right of the Event window.
[6]
　The "Insert Event Command" window will appear.
　Confirm that the top tab shows "Show Message".
　
　・In the text input field in the center,
　　type "Yuichi attacked!" and
　　click the "input" button.
　
[7]
　The event is now complete.
　Does the completed event settings look like the following?

　1: Graphic is "chicken"
　2: Movement Route is "Approach Player"
　3: Move Frequency is "0: Every Frame"
　4: Trigger Condition is "Event Touch"
　5: The Event Command display on the right shows:
　　　■Message: Yuichi attacked!
　　　■
[8]
　Now save the map and
　test the event you created.

　After clicking "Save (Entire Map)",
　click "Test Play".
[9]　
　It's hard to tell from a still image, but did
　"Yuichi attacked!" display the moment
　Yuichi came at you and made contact,
　as shown in the image on the right?

　The "Event launched when the opponent touches you" is now complete.
　Good work!
　



[Side Note]
　As-is, even after contact the chicken keeps chasing you,
　so if you don't move, the event will trigger repeatedly.

　For an enemy symbol that disappears after battle occurs and is defeated,
　where the event does not trigger a second time after occurring once,
　see:
　◆ Event that disappears after the event and comes back when you return to the map
　◆ Event that never reappears once it has disappeared
