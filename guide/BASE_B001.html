【[Battle] I want to create a combat command like "Attack" or "Defend" that triggers a skill immediately when selected】

[Goal]
　We will add a command "Use Healing Magic" to the player character Wolfar (the blue person) in the sample game,
　so that selecting it uses the "Heal" magic.
　
　Note: if you want to add a technique other than an existing one, replace the parts referring to "Heal" in the description below with your own technique.
　For how to add new techniques (magic), refer to the I want to add more skill (magic) types section.

【Overview of command creation】 　To create a command and have a player character (ally) use it, there are two main steps:
　1: Add the command in the User Database
　2: Configure the command for the player character (ally) in the Changeable Database
　
　First, let's add the command in the User Database.

[1]
　Clicking User Database will
　display a window like the one on the right.
　　
[2]
　The "Type" setting is probably "0:skill",
　so please change it to "6:Combat Command".
 →
[3]
　Looking to the right, you will see a list like the one below.
　Let's configure a new command in the unused slot "9:".
　(In the figure below, "Use Healing Magic" has been entered as the command name,
　so "9:" has become "9:Use Healing Magic" in line with the name.)
　
　As shown in the figure below, enter "Use Healing Magic" as the command name,
　"Uses "Heal"" as the command description,
　and since this command uses "Heal", set the effect when the command is selected to
　"[6]:Heal [User0]".
　

　
　Note: if you want this command to be disabled by "skill (magic) seal" just like normal magic,
　change the command type from "[0] Normal" to "[1] Disabled by Special Skill Seal".

　This completes the User Database settings.
　Next, we will change the Changeable Database so that Wolfar (the blue person) can use this command.

[4]
　Clicking Changeable Database will
　display a window like the one on the right.
　　
[5]
　Confirm that the "Type" is set to "Player Character Status".

　(If it is not,
　　click "0:Player Character Status")
[6]
　Looking to the right, you will see a list like the one on the right.
　Wolfar should already be selected.
　
　Next, change to Page 3 using the buttons at the top
　labeled "Page 1", "Page 2", etc.
　The list will change to look like the image on the right.
　
　Here, click the 45th entry "[Command 4]"
　which currently shows "[-1]: Empty",
　and change it from the displayed menu to the command you just created:
　"[9]:Use Healing Magic [User6]".
[7]
　The Changeable Database command settings are now done.
　
　Running a Test Play, the
　"Use Healing Magic" combat command is now displayed successfully.
