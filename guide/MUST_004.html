[I want to change the game's starting position]

[Goal]
　In the sample game, the game starts from the title screen. We will change the setting so that
　the game begins directly in front of the Professor in "Sample Map A".

　　 　
　　　↑We will set the spot where the blue character is standing as the game start position.

[1]　Load "Sample Map A" — the starting map

　Open the Map Selector window and
　select "1: Sample Map A".

　
[2]　Click the icon labeled "[Ev]"
　　　to switch to Event Layer editing.
[3]　
　Right-click the spot where you want the game to start
　(here, in front of the Professor),
　and select "Set as Game Start Position".
[4]　
　A window titled "Register Destination" will appear.
　Do not change anything and just press "OK".
[5]
　Running Test Play will now start the game at the location shown on the right.


　The game start position has been set successfully!



[Side Note] How to revert the start position?

　　Switch the map to "0: Title", then
　　right-click anywhere and select "Set as Game Start Position".
　　The rest is the same as above.


　※The event in the sample game's "Title" map uses
　　the "Auto-execute" trigger,
　　so the opening event runs no matter where the player is on that map.
