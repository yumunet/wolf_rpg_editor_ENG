【Event launched by Confirm key】

[Goal]
　Create an event that launches when the Confirm key is pressed.

　Here, inside "Sample Map A" in the sample game,
　create a new event for a treasure chest that displays the message "There is a treasure chest." when examined.

　
　Something like this.

　※ The procedure is almost the same as ◆ I want to create a new event,
 　　but here we will create the event by selecting the graphic from a tile chip rather than a character chip.

[1]
　・First, use the Map Selector to switch to "Sample Map A".
　
　・Create a new event at a suitable location.

　→ If you are unsure, refer to steps [1]～[3] of ◆ I want to create a new event.
[2]
　Let's set the event's graphic.
　
　Double-click the area enclosed in a red box.
[3]
　A window like the one on the right will appear.
　
　・From the list on the left side of the window,
　　select (Tile Set).

　・The map chips will then display on the right.
　　Select the treasure chest from among them.
　　The treasure chest is located near the bottom.
　　Use the scroll bar on the right to find it.
　
　※ In (Tile Set), you can only select from the
　　　tile set currently used on the current map.
[4]
　The treasure chest has been set as the event graphic.
　Next, confirm the "Trigger Condition".

　・Pay attention to the area circled in red.
 　You can set the "Trigger Condition" here.

　・Click to display 5 Trigger Conditions,
 　and select "Execute on Confirm key".

　※ Newly created events are already set to
　　"Execute on Confirm key" by default,
　　so this time we just need to confirm it.

　This completes the setting of making the current event
　an "event launched by Confirm key".
[5]
　Finally, create the event contents.
　
　Click "■ Show Command Window ■"
　at the bottom right of the Event window.
[6]
　The "Insert Event Command" window will appear.
　Confirm that the top tab shows "Show Message".
　
　・In the text input field in the center,
　　type "There is a treasure chest." and
　　click the "input" button.
　
[7]
　The event is now complete.
　
　Is it in the following state?
　Pay attention to the area enclosed in the red lines.

　1: Graphic is "treasure chest"
　2: Trigger Condition is "Execute on Confirm key"
　3: The Event Command display on the right shows:
　　　■ Message: There is a treasure chest.
　　　■
[8]
　Now save the map and
　test the event you created.

　After clicking "Save (Entire Map)",
　click "Test Play".
[9]　
　Approach the created event and
　press the Confirm key (Enter / Space / [Z]).

　Did "There is a treasure chest." display
　as shown in the image on the right?

　The "Event launched by Confirm key" is now complete.
　Good work!
