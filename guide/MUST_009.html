[I want to try creating a new Common Event]

[Goal]
　Create a new Common Event
　and call it from a map event.


[1]
　First, open the Common Event list.
　Left-click  shown in the circle on the right.
















[2]
　When the Common Event Editor opens,
　scroll down in the Common Event list
　on the left side of the window
　to find the section labeled
　"■ Free to use below ■".

　It is quite far down, so just
　drag the scrollbar all the way down.

　1. Common Events in this area and below
　are ones you can freely modify.



















[3] I ran out of blank slots!
　If you run out of empty Common Event slots,
　click the area outlined in the box.

　A "Set Data Count" window will appear.
　Enter a number larger than the current value
　to increase the maximum number of Common Events,
　giving you more empty slots.

　Conversely, reducing the number will permanently
　destroy the Common Events in those removed slots,
　so be careful.



















[4]
　Now let's select an empty Common Event slot
　and create a Common Event.

　1. Select the Common Event to edit.
　   We'll use slot 216 this time.

　2. Common Event Name
　   Give the Common Event a name.
　   Choose something easy to identify.
　   We'll call it "Display Message 01".
　   ※ If left blank,
　   you won't be able to call this Common Event
　   by name, so be sure to name it.

　3. Trigger Condition
　   Since we will call it from a map event,
　   select "Call Only".



















[5] Show Message
　1. Click Show Event Command Input Window
　   to open the Event Command window.

　2. Select 1 Show Message.

　3. Type the message you want to display.
　   This time, enter:
　   '"Display Message Common 01" Test'

　4. Click Input to finish.

　Your new Common Event is now complete.



















[6] Create the map event
　· Open the Map Selector  and switch to the map where you want to create an event.

　· Create an event somewhere on the map and set the chicken image for it.

　For the steps, refer to [1] through [5] of ◆An event that activates with the Confirm key.



















[7]
　Once created, click
　"■ Show Command Input Window ■"
　at the bottom-right of the Event window.

1. When the "Insert Event Command" window appears,
　 click and select I Common Event.

2. Confirm that "Call by Common Ev Name" is selected
　 (the radio button next to it should be filled in).
　 ※ If it's not selected, click it to select it.

3. Click the area outlined in box 3 to see
　 a list of available Common Events.
　 Since we want to call the new Common Event,
　 select "Display Message 01" that we just created.



















[8]
　There is nothing else to configure,
　so click Input.

　Once input, the event setup is complete.



















[9] Test Play
　Run Test Play and try talking to
　the event you just created.

　If the message '"Display Message 01" Test' appears,
　you succeeded!

　Great work!

　※ If it doesn't work, check:
　1. You may have selected the wrong Common Event in "I Common Event".
　2. The Common Event you created may have no commands in it.
