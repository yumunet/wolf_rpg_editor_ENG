[I want to set the level at which characters learn skills]

[Goal]
Configure Wolfar to learn "Power Charge" at level 5 in the sample game.

[1]
Click the Changeable Database
to open the window shown on the right.
[2]
Select "1: ┣ Skill Learn Level".
[3]
Looking to the right, you will see input fields
for the learn level and the skill to learn.
[4]
Set the second "<Ability> Learn Level" to 5,
and the "┗ Skill to Learn" below it to "[2]: Power Charge".

※ The first entry already has "Heal" set to be learned at level 3,
so leave that one as-is.
[5]
When done, click "OK" at the bottom-right of the window.
[6]
Run Test Play .

If Wolfar learns "Power Charge" when reaching level 5, it worked!

Great work!
