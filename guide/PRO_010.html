【Tips for movement key input [1–9]】

[1] Calculating horizontal and vertical movement amounts from key input commands

■ A method for obtaining the movement key input or facing direction value, then calculating the horizontal movement amount and vertical movement amount from it.
※ If the value is 4, it means left, so horizontal movement amount [-1], vertical movement amount [0].
　If the value is 9, it means upper-right, so horizontal movement amount [1], vertical movement amount [-1].
　Please understand the above before using this.

Subtract 1 from the obtained movement key / facing direction value. This gives:
　　789　　　678
　　4  6　→　3  5
　　123　　　012
From here, take %3 (remainder when divided by 3):
　　678　　　012
　　3  5　→　0  2
　　012　　　012
Then subtract 1 to get a usable value [-1 to 1] as the horizontal movement amount.

For /3 (integer division by 3):
　　678　　　222
　　3  5　→　1  1
　　012　　　000
If dividing by -3 instead of 3, the result is [-2 to 0],
and adding 1 gives a usable value [-1 to 1] as the vertical movement amount.




















[2] Calculating movement key input / facing direction value from horizontal and vertical movement amounts

※ Assume the range of each movement amount is [-1 to 1].
First, add the horizontal movement amount [-1 to 1] to the value 5:
　4　←　5　→　6
　-1　　  0　　  +1　　[horizontal]
Then add the vertical movement amount multiplied by -3:
　　　　　　[vertical]
　７┃８┃９　 -1
　↑┃↑┃↑
　４┃５┃６　  0
　↓┃↓┃↓
　１┃２┃３　+1
This gives the calculated value.




















[3] Calculating the opposite facing direction value from a facing direction value

Opposite facing direction = 10 - facing direction [1 to 9]

Facing lower-left [1]: 10 - 1 = 9 [upper-right]
Facing up [8]: 10 - 8 = 2 [down]




















[4] Calculating whether movement key input / facing direction value is diagonal [1, 3, 7, 9]

Take the remainder when dividing the value by 2 to get either 0 or 1.
Odd numbers [1, 3, 5, 7, 9] have a remainder of 1 when divided by 2.
Even numbers [2, 4, 6, 8] have a remainder of 0 when divided by 2.
Therefore, if the remainder is 1, it is diagonal.
