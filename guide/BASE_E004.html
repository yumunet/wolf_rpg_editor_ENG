【I want to increase/decrease held money】


[Goal]
We will create an event that increases or decreases held money.
Specifically, it is an event where examining a treasure chest gives the player money.

[1]
First, referring to ◆ An event launched by the Confirm key, create a map event.

[2]
Next, click "■ Show Command Input Window ■" at the bottom right of the Event window.

[3]
When the "Insert Event Command" window appears,
click the Other 2 tab. (Image 1)

Once the tab switches,
click the Insert Event section
and select
"Common 3: ○ Money Increase/Decrease".
(Image 2)

[4]
Look at the Common Event Input (Value). ([3] Image 3)
Enter the amount to increase or decrease in Money Change Amount.
※ A negative value (-) decreases the amount.

Clicking the Acquisition Message field lets you choose whether to display a message when money changes.
※ Such as "Obtained ○○ Gold.", etc.

When done, click the Input button ([3] Image 4).
The event is now complete.

[5]
Let's save before we forget.

[6]
Finally, click Test Play to confirm the event works.
The image shows the case where Acquisition Message is "Yes".
If you set it to "No", press X to open the menu and check from there.

This concludes "I want to increase/decrease held money".
Good work!




[Side Note]
As-is, the player can obtain money every time they examine the event.
For how to create an event that only grants money once, see ◆ A treasure chest that stays open once opened.
