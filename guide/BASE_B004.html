【◆[Battle] I want to create a status ailment】

[Goal]
We will create a new status ailment "Flinch" (unable to act for 3 turns).

This is a bit long, but broadly speaking, the following 3 things are covered:

　☆ Create a new "Flinch" entry in User Database "8: Status Ailments".
　☆ 『Special Skill Section』: Create a special skill "Intimidate" that inflicts
　　　　　　　　　　　　　　　the "Flinch" status on an enemy, and teach it to the player character.
　☆ 『Weapon Section』: Create a weapon "Impact Dagger" that inflicts the "Flinch" status
　　　　　　　　　　　　　　　on an enemy with normal attacks when equipped, and equip it on the player character.

The two sections below (『Special Skill Section』 and 『Weapon Section』) are optional.

☆ Let's first create the status ailment "Flinch".

[1] Clicking the User Database will
display a window like the one on the right.
[2] When the window first opens,
the "Type" is set to "skill".
Please click "8: Status Conditions".
[3] Click on an empty slot in the "data" list.
[4] Enter values for the "Page 1" entries.

　Set "Status Name" to "Flinch",
　"Minimum Duration [-1=Permanent]" to "3",
　"Text on occurrence [(name)~]" to "flinched!",
　and change "Action Restriction" from "No Restriction" to "Unable to Act".
[5] This completes the status condition settings.
Click "OK" at the bottom of the window to close it temporarily.
This saves the work you have done so far in WOLF RPG Editor.

Next, create a means to trigger the status ailment.

☆ 『Special Skill Section』: If you want to inflict the "Flinch" status on an enemy
　　　　　　　　　　　　by using a special skill, continue below.
☆ 『Weapon Section』: If you want to create a weapon that inflicts the "Flinch" status
　　　　　　　　　　on an enemy via normal attacks when equipped, click here.
☆ 『Special Skill Section』


[1] Clicking the User Database will
display a window like the one on the right.
[2] Confirm that the "Type" is set to "skill".
(If it is not, click "0:skill")
[3] Click on an empty slot in the "data" list.
[4] Enter values for the "Page 1" entries.
Set "Skill Name" to "Intimidate",
"┏ Effect Target" to "10: One Enemy [Attack]",
and "Text on use [(name)~]" to "intimidated the enemy!",
respectively.
[5] Next, enter values for the "Page 2" entries.
Set "Status to Inflict" to "Flinch".
[6] Press the Update button or OK button.
The new skill "Intimidate" has been saved.
Next, teach it to the player character so it can be used.
[7] Clicking the Changeable Database will
display a window like the one on the right.
[8] Select "1: ┣ Skill Learn Level".
[9] Looking to the right, you will see
input fields for learn levels and
skills to learn, as shown in the figure on the right.
[10] Set the second "<Ability> Learn Level" from the top to "1",
and set the entry one below it "┗ Skill to Learn" to "Intimidate".

※ The topmost entry is already configured to learn "Heal" at level 3,
so leave that as is.
[11] Once configured, click "OK" at the bottom right of the window.
The player character has now learned the special skill "Intimidate".
When used in combat, the enemy will be inflicted with the "Flinch" status,
and the enemy should be unable to act for 3 turns.

This completes the creation of a status ailment using a special skill.
Good work!

(Below this is the method to create a status ailment using a weapon.)
☆ 『Weapon Section』


[1] Clicking the User Database will
display a window like the one on the right.
[2] When the window first opens,
the "Type" is set to "skill".
Please click "3:weapon".
 　→　
[3] Click on an empty slot in the "data" list.
[4] Enter values for the "Page 1" entries.
Set "Weapon Name" to "Impact Dagger",
"Equipment Type" to "0: Dagger [User5]",
and "Attack Animation" to "1: Big Slash [User1]",
respectively.
[5] Enter values for the "Page 2" entries.
Set "┏ Status to Inflict" to "Flinch"
and "┗ Infliction Rate" to "50".
[6] Press the Update button or OK button.
The new weapon "Impact Dagger" has been saved.
[7] Clicking the Changeable Database will
display a window like the one on the right.
[8] Confirm that the "Type" is set to "0: Player Character Status".
(If it is not, click "0: Player Character Status")
[9] Configure the Page 2 entries for
data 0 "0:Wolfar".
Change the 6th entry from the top, "Equipped Weapon",
to "Impact Dagger".
[10] Press the Update button or OK button.
Wolfar's weapon is now the "Impact Dagger".

Enter combat without changing the weapon and try a normal attack.
If configured correctly, there will be a 50% chance of inflicting the "Flinch" status on the enemy,
making the enemy unable to act for 3 turns.

That covers how to configure a status ailment using a weapon.
Great work! You're done.
