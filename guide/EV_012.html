【I Want to Change the Position Where Choices Are Displayed】

　[Goal]

　In WOLF RPG Editor, choices are displayed in the center of the game screen.
　However, quite a few games display their title screen's New Game/Continue choices
　in a position other than the center, such as the bottom-right.

　This time, by creating an event that displays choices in the bottom-right of the screen,
　we will learn how to change the display position of choices.

　【Steps】
[1] Create the event

　Create an event somewhere on "Sample Map A" and
　set the chicken image.

　If you're unsure how, please refer to steps [1]～[5] of
　"I want to create a new event".
[2] Enter Event Command: X coordinate of the choice

　Click "■ Show Command Window ■"
　in the bottom-right of the Event window.

　Right away, before entering the choice itself,
　we will enter the command that changes the display position of the choice.

　The position where the choice is displayed is specified by coordinates on the screen.
　WOLF RPG Editor's game screen is a plane with X and Y axes, similar to a graph in mathematics,
　except that the Y coordinate is reversed — the further down you go, the larger the number. Please note this.
　The coordinate at the very top-left of the screen is (x, y) = (0, 0) — the origin —
　and the coordinate at the very bottom-right is (x, y) = (320, 240).

　Now let's enter each coordinate.
① Switch the Event Command Input window tab to
　"3 Variable Operation",
② and select "Sys" using the button for the variable type.
③ From the dropdown list, select "Sys3: Choice Window X coordinate",
④ and enter "250", "+", "0" on the right side.
⑤ When done, click the "Input" button at the bottom-right of the window.

　This sets the X coordinate to 250.
[3] Enter Event Command: Y coordinate of the choice

　Next, configure the Y coordinate in the same way.
① From the dropdown list, select "Sys4: Choice Window Y coordinate",
② and enter "160", "+", "0" on the right side.
③ When done, click the "Input" button at the bottom-right of the window.

　This sets the Y coordinate to 160.
　The settings to change the choice display coordinate to (x, y) = (250, 160) are now done.
[4] Enter the choice

　All that's left is to enter the choice and do a Test Play.
① Switch the Event Command Input window tab to "2 Choice",
② and enter "Choice 1" and "Choice 2" in the top two input fields respectively.
③ When done, click the "Input" button at the bottom-right of the window.

　The Event Commands to display choices in the bottom-right of the screen are now complete.
[5] Confirm

　Let's do a Test Play. When the event launches,
　the choice should be displayed in the bottom-right of the screen.
　If something goes wrong, review the order
　of the entered Event Commands.
　The Variable Operation commands for the window coordinate
　must be entered before the choice for them to work correctly.
　Great work!
　[Side Note]
　・Regarding settings around choices

　When a choice is launched, the cursor starts at the top choice,
　but you can change this so the cursor starts on a choice
　further down.
　Using Variable Operation, if you set the right side of Sys7: Choice next initial position (0, 1, 2...)
　to, for example, 1 and enter it, (since choices are numbered 0, 1, 2... from the top)
　the choice will launch with the cursor on the second choice from the top.

・Also, if you set the right side of Sys73: Text/Choice mouse input (1 = accept)
　to 1 and enter it, choices can be selected with mouse controls.
　This is an essential feature for games centered around mouse operation.
