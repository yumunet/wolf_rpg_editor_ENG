【I want to add a custom command to the menu】

　[Goal]

　The "Talk" command on the menu in the sample game
　is configured in the User Database to call a Common Event.

　In this topic, we will create a Common Event that loads save data,
　then configure it in the User Database to be called as a "Load" command from the menu,
　and learn how to configure a specific Common Event to be
　callable from the menu screen.

　【Steps】
■Creating the Common Event "Load"

　First, create the Common Event that we will call later.
[1] Creating the Common Event

　Create a new Common Event.

　① Click Common Event to
　　open the Common Event window.
　　Select a Common Event number that has no event in it yet.
　② Click "Show Command Input Window" at the bottom of the window.
　③ Change the tab to "I Common Event" and configure the event operation process field.
　④ Select "Insert Event" with the radio button
　　(this is selected by default),
　⑤ Then select "Common 14: ○ Various Menu Calls" from the dropdown on the right.
[2] Entering the Common Event content

　Next, specify and enter the content of this event.
　We will add an Event Command to load save data.
　In the 【Common Event Input (Value)】 field,
　set "Menu to call" to "-1: Load Screen".
　Once configured, press the "Input" button at the bottom right of the window.
[3] Entering the Common Event name

　The process has been created. Let's give this Common Event a name.
　Enter "Load" in the "Name" field at the top of the Common Event window,
　then press "OK" at the bottom of the window.
　This creates a Common Event that "shows the Load Screen when called".
■Configuring "Load" to appear in the menu
　Click User Database to
　open the User Database window.
　① When the window first opens, the "Type" is set to "skill".
　　Please click "17: System Settings".
　② In "Page 1", for the "Menu Column Commands" entries 1–8,
　　enter "[11] Special Menu B" in an empty slot.
　　If Special Menu B is already in use,
　　use a different special menu (whichever of A, C, or D is unused).
　③ For the entry below, set "Special Menu B Name" to "Load",
　　and "┗ Special Menu B Call Common" to "Load [Common Event]".
　　(If you chose a special menu other than B in ②,
　　configure that special menu's entry instead.)
　④ Press the Update button or OK button.
"Load" has now been added to the menu.
■Test Play
　The "Load" option should now appear at the position
　you specified in the menu's User Database settings.
　Run the game and confirm that the Load Screen is called.

　Setup is complete. Good work!
　[Side Note]
Once you know how to do this, as long as you can prepare a Common Event,
you can do almost anything.

For example, if you create a new Common Event that calls
Common Event "Common 028: ◆ Battle Trigger" and register it in the
User Database to be called from the menu, you can fight a specific enemy character
anytime and anywhere, or even hold combat practice sessions against party members.
You could also create a Common Event that calls everything from
"Common 33: [1] Shop Initialization" through "Common 35: [3] Shop Process Run"
together, giving you access to a shop with the same inventory anytime, anywhere.
