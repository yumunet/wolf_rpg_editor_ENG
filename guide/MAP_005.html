【Draw Auto-Tile Borders Freely】

[Goal]

Auto-tiles are tiles that automatically connect when placed next to each other, but
drawing them exactly the way you want may take some practice.

This time, let's try drawing an unusual map.
Those who have tried making maps may think "Huh?" at first.
But with the method you'll learn today, you can achieve this kind of drawing on a single layer!

Note!
The method differs between Ver1.31 and earlier versus Ver2.00 and later.
For the method in Ver1.31 and earlier, see Appendix 2.
You can even write text like this!

【Steps】

[1] Open "Sample Map B" in the sample game.

This time, as shown in the example, we will try adding a river to the right of the river already to the right of the goddess statue.

First, select layer 2.
Try placing water auto-tiles like in the example. Then...
I want to draw it like this

[2] The tricky part of auto-tiles

As you see on the right, all the water areas connect together.
Auto-tiles are drawn connected when the adjacent tile has the same auto-tile,
and drawn disconnected otherwise.
Also, the edges of the map are treated the same as if an auto-tile were present there.
Oh no, they all connected!

[3] The ultimate technique: "Auto-tile Disable"!

The useful technique here is "paste while holding the Shift key".
Using this, you can copy the exact shape of a part as it appears on the map!
It has tremendous power when used with auto-tiles!

We will call the method described here "Auto-tile Disable".
This method is actually used in certain other tools as well (and many people there don't know about it either).

[4] Let's try it in practice!

First, right-click on the part you want to copy
to copy it.
Then, at the location where you want to place it, try "Shift + left-click".
Then...
Copy this part

[5] How does it look?

A part with the same shape as the copied part is pasted!
From here, right-click to copy, then Shift + left-click to paste
the part in the shape you want.
There you go! It's pasted in exactly the same shape!

[6] To exit this mode

While in this mode, auto-tiles placed will not connect automatically.
To exit this mode, simply "left-click only" (without Shift).

However, note that if you place a chip normally next to an auto-tile created with "Auto-tile Disable",
the shape of that auto-tile may change.

Appendices

【Appendix 1】 Number of auto-tile patterns

Auto-tiles are composed of roughly 5 types of parts.
From these, various patterns are created — but how many patterns are there?

The answer is... 47 patterns! (Including "none", it rounds up to 48.)
All of those patterns are created from just those chips.

The patterns are shown in the figure to the right.
The left side shows the original auto-tile shape; the right side shows all patterns it creates.

【Appendix 2】 What to do in Ver1.31 and earlier?

The above method cannot be used in versions Ver1.31 and earlier.
In that case, please try the following method.

First, Shift + right-click on the auto-tile with the shape you want to copy it.
Then, at the position where you want to place it, left-click!

This will place the auto-tile in the same shape as the one you copied.
To return to normal auto-tile placement mode,
press Shift + left-click again to return to the original mode.

You can also distinguish between these modes as follows:
Select an auto-tile from the chip selection screen and draw it on the map:
If this appears, you are in "normal" mode.
If this appears, you are in "Auto-tile Disable" mode.
