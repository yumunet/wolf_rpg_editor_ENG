【I want to properly set up the Game Over process】

[Goal]
　　　We will create a process where, when all allies' HP reaches 0 during combat,
　　　"Game Over" is displayed, the screen fades to black, and the game returns to the title screen.

[1] Open "Common Event Settings" ,
　　　select Common 82, and open it.
[2] Next, click "Show Event Command Insert Window"
　　　at the bottom of the Common Event editor.
[3] In the "Insert Event Command" window,
　　　select the "Screen Process" tab, click "Fade to Black" under Screen Color Adjustment,
　　　set the process time to 10 frames, and press Input.
[4] Delete the text that was there from the start using the Backspace key,
　　　then enter "Game Over!".

Refer to [8][9] of I want to try creating a new event.
[5] Since the screen color adjustment takes 10 frames,
　　　enter a Wait of 10 frames from the Other 1 tab.
[6] As the process to return to the title, from Other 1,
　　　press "Go to Title Screen".
[7] From "Screen Process", click "Reset Color" under Screen Color Adjustment,
　　　and press Input.
[8] Next, open the "User Database Editor" .
[9] Open "15: Basic System Settings" and press "Page 3" .
Confirm that the [Combat] Game Over Process Event is set to Common 82,
then press "OK".

That's how to create the Game Over process. Great work!
