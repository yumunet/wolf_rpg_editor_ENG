【I want to trigger a battle】

[Goal]
　In "Sample Map A" of the sample game, we will create
　a chicken event that triggers a battle against "3 Goblins".

[1]
　• First, use the Map Selector to switch to "Sample Map A".

　• Create an event somewhere suitable and
 set it to the chicken image.
　　(Placed at the position shown in the image on the right.)


 → If you are unsure, follow
　  ◆ I want to try creating a new event
　　steps [1] through [5].
[2]
　Click "■ Show Command Window ■"
　at the bottom right of the Event window.
[3]
　When the "Insert Event Command" window
　appears, select the tab at the top
　labeled "Other 2".
[4]
　Select "Insert Event" and
　choose Common 24 "◆ Battle Trigger".
[5]
　Next, configure the "Common Event Input (Value)" fields as follows.

• Enemy group to appear: [1: 3 Goblins]
• Can flee: "0: Can Flee"
• On defeat process: "0: Game Over Event"
 ※ There are two options [0: Game Over Process / 1: Continue Event],
 but here we set it so that losing
 triggers the Game Over event.
• BGM: "3: Combat BGM"

　Once configured, click "Input".
[6]
　If it ends up looking like the
　image on the right, you have succeeded.
[7]
　Now let's save the map and test the event.

　Click "Save (Entire Map)", then
　click "Test Play".
[8]
　Start the Test Play and
　press the Confirm key facing the chicken
　— a battle against 3 Goblins now starts!


　This completes a simple combat event
　that triggers every time you interact
　with the chicken.
　Good work!
