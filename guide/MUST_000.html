[I want to install WOLF RPG Editor]

[Goal]
　Extract the downloaded WOLF RPG Editor archive,
　then launch the sample game and open the editor.


　※This guide assumes you already know how to download files
 　　and perform basic file operations (such as deleting files).

[1]
　We start with the WOLF RPG Editor archive file
　"WolfRPGEditor_???.exe"
　already downloaded.
　(??? is a version number, such as "116")

　First, run the downloaded file.
[2]　
　You will see "Please specify the extraction folder",
　so select "OK".

※You may be asked whether to allow execution first,
　 just select "OK" to proceed.
[3]
　When extraction is complete,
　a "WOLF_RPG_Editor" folder will be created.

　At this point, the original file
　"Wolf_RPGEditor_???.exe" is
　no longer needed, so delete it.

　After deleting it, open the "WOLF_RPG_Editor" folder.
[4]　
　The contents of the "WOLF_RPG_Editor" folder
　look like this.

　There are many files, but for now
　the only ones you need are
　Game.exe and Editor.exe.

 ※By the way, all files used by the game are
　inside the "Data" folder,
　so be careful not to accidentally delete it!

※The "Data Collection" folder contains useful
　data. You can safely delete it when
　distributing your game.
[5]　If you want to play the sample game

　Running "Game.exe" lets you
　play the sample game.

　A screen like the one on the right will appear and the game starts.
　On the keyboard, Z (Enter/Space) is Confirm,
　X (ESC/Backspace) is Cancel,
　and Shift is the Sub key.

　To exit, press the "×" in the top right,
　or press Alt+F4.
[6]　If you want to edit

　Running "Editor.exe" lets you
　start editing.

　By default, the editor opens to Map 0 "Title".
　It is a solid blue map with
　just a single event on it.


　If you played the sample game in [5],
　try switching maps to find the familiar
　"Sample Map A"!
