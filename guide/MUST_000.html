◆ Installing WOLF RPG Editor

[Goal]

　Download and extract WOLF RPG Editor, then run
　the Sample Game, and open the Editor.


　※ This guide assumes you already know how to download files
 　　and perform basic file operations (such as deleting files).

[1]　Downloading the program

　You can download the latest version of the Editor  
　from ModDB. After that, you will see a zip file
　with a filename similar to this:

　"en-US Woditor vx.xxx.zip" (where xxx
　is a version number, such as "3.691")

　First, extract the downloaded file.
[2]　Extracting the program

　A window will appear asking you to select an
extraction folder. Select "OK".

※ You may first be asked whether to allow the program  
　 to run. In that case, select "OK" as well.
[3]　After extraction is complete

　Once extraction is complete, a folder named:        
　"en-US Woditor" will be created.

　At this point, the original file
　"en-US Woditor vx.xxx.zip" is no longer
　needed, so you can safely delete it.

　Now you can open the extracted folder.
[4]　WOLF RPG Editor files

　Inside the "en-US" folder you will see the files.    
　
　There are a lot of them, but for now the only
　ones you need are: Game.exe and Editor.exe.

 ※ All files used by the game are stored inside the
　"Data" folder, so be careful not to delete it!

 ※ The "Extras" folder contains useful sample resources.
　You can safely delete it when distributing your game.
[5]　If you want to play the sample game

　Run "Game.exe" 　

　The sample game will start, and a screen like        
　the one on the right will appear.  

　Controls:
　Z (Enter/Space): Confirm,
　X (ESC/Backspace): Cancel,
　Shift: Special Action Key.

　To exit the game, click the "×" in the top right
　corner, or press Alt+F4.
[6]　If you want to start editing

　Run "Editor.exe"
　The editor will launch.

　By default, you will see Map 0 "Title", which is     
　just a plain black map with a single event and
　an "S" start position marker shown as a small
　square.


　If you played the sample game earlier in
　step [5], try switching maps and opening
　the familiar "Sample Map A"!
