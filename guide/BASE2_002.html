【I want to set an element on a weapon】

　[Goal]

　If you play RPGs frequently and are already familiar with elements, this may be nothing new,
　but elements are rules that determine what a character is strong or weak against,
　adding strategic depth to the game.

　In the WOLF RPG Editor Basic System, you can set elements on allies, enemies, equipment, and skills.
　With elements, you can create monsters that are weak to fire but strong against ice,
　skills that can attack all enemies with an earthquake but are immune to flying monsters, and even
　weapons like a "Holy Sword" that deals massive damage only to a specific boss like the "Demon King".

　In this topic, we will set the fire element on the weapon "Dagger" that is already included in the sample game,
　and learn how to configure elements on weapons.

　【Steps】
　In this topic's 【Steps】, we will first configure the "Element Settings" section
　so that the Dagger has the fire element,
　then in the "Confirm Settings" section we will show you how to do a Test Play.
　■Element Settings
[1] Opening the User Database

　Clicking the User Database will
　display a window like the one on the right.
[2] Selecting the type to edit

　When the window first opens, the "Type" is set to "skill".
　Please click "3:weapon".
[3] Selecting the data to edit

　There is a data list in the middle of the window.
　Please select "0:Dagger" from this list.
[4] Editing the entry

　① Switch the page to "Page 2".

　② For entry number 28 "Granted Element 1", select
　　"[1] Fire Attack [User7]".

　(Entry number 29 "Granted Element 2" can also be configured,
　　so you can attach up to two elements to a weapon.)

　③ Once configured, press the Update button or OK button to save.

　The "Fire Attack" element has been added to the Dagger.
　That completes the setup for a weapon with an element.
　■Confirming the Settings
　The enemy "Dragon Vine" included by default in the Basic System
　is configured to be weak against fire attacks.

　If you compare the Dagger with the fire element to the Dagger without any element,
　you should see a change in the amount of damage dealt.

　If you want to do a Test Play but aren't sure how to create a combat event,
　first refer to "I want to configure an enemy group"
　to create an enemy group containing Dragon Vine in the User Database,
　then refer to "I want to trigger a battle"
　to call it from a map event or similar.

　Simply adding an element shows few visible differences beyond the change in damage,
　so you may want to change the name and graphic to give it your preferred presentation/effect.
　Good work!
　[Side Note]
■Giving a weapon a status ailment infliction effect

　In addition to being able to set elements, User Database "3:weapon" also lets you
　add the ability to inflict status ailments such as poison or paralysis.

　Entry number 18 "Inflicted Status Condition" and entry number 19 "Infliction Rate [%]"
　are the relevant settings.

　These let you configure the type of status ailment inflicted on the attacked target
　and the probability of inflicting that ailment, respectively.
