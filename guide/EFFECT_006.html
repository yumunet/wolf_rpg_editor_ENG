【Other things you can do in the message window】

【Question】 What else can you do inside a message?
In the EFFECT section, we have described various operations available in "Show Message".
Let's look at what other things can be done inside messages here.
Please also refer to the following topics already described:

  • "◆ I want to make text in a message larger (\f[xx])"
  • "◆ I want to change text color in a message (\c[xx])"
  • "◆ I want to change the font in a message (\font[x])"
Here we describe items other than those.

【Solution】 Here is what you can do

　　Using special characters, you can do the following. Methods and examples are described in 【Details】.
  • Set the maximum font size for a line
  • Add an outline to text
  • Add anti-aliasing to text
  • Change the text display speed
  • Condense or expand character spacing
  • Change the line spacing in a message
  • Move the display coordinate of text
  • Add ruby (furigana) to text
  • Display an icon inside a message
  • Display text centered or right-aligned
  • Wait a fixed time during message display before the next display
  • Pause message display partway through
  • Display the entire message at once
  • Erase a message without waiting for key input
  • Other notes

★Note!
When using special characters, uppercase and lowercase letters are distinguished! Be careful when entering them!

【Details】 Here is how to do it

■　Set the maximum font size for a line
Use \m[xx].
This determines the maximum font size displayed on that line.
(Positions are aligned based on this font size as the reference.)
Use it at the beginning of the line.
It is useful when the display is uneven due to multiple font sizes or icons being used on the same line.
The effective range is from \m[xx] to the end of the line.

★Note!
Up to Ver2.02a, when the screen size is 320×240,
if there are characters of different sizes, they will be center-aligned.
From Ver2.10 onwards, they are aligned at the bottom edge.
Text alignment is tidied up.
■　Add an outline to text
Use \E.
Text displayed after this will have an outline (edge). It looks like a shadow effect.
(The outline color is fixed to black; system variable 21 can change the outline position.)
To remove the outline, use \N. This is the default state.
If you don't specify \N to remove it, the effective range extends from \E to the end of the message.
To apply it to a specific part, surround the text like "\Etext\N".

For example, use it on parts you want to emphasize.
Title text with this kind of decoration might look nice.
With an outline it stands out, right?
■　Add anti-aliasing to text
Use \A+.
Anti-aliasing is applied to the text, making it display more smoothly.
To remove anti-aliasing, use \A-.
When the font size is small, or when using a font where the blocky look is a feature,
it is better not to have anti-aliasing.
If you don't specify \A- to remove it, the effective range extends from \A+ to the end of the message.
To apply it to a specific part, surround the text like "\A+text\A-".

For example, use it when the jagged edges of text bother you.
It may be worth trying for large title text displays.
Large text looks natural too, but for small text the effect can be reversed.
■　Change the text display speed
Use \sp[xx].
The Show Message display speed after this becomes xx characters per second.
(This does not affect system variable 9, etc.)
The effective range is from \sp[xx] to the end of the message.

Note that values of 0 or less result in the same behavior as instant display.


For example, it can be used as an accent in message display — making a character talk rapidly, or emphasizing a word.
Using it too much may disrupt the reader's pace,
so it is recommended to use it only at key moments.
Speeds up and slows down.
In this example, circles display at 2 characters per second, and stars display instantly.
■　Condense or expand character spacing
Use \-[xx].
The character spacing is condensed by the number of pixels specified by xx.
This can be used when character spacing is too wide.
(Over-condensing can cause characters to overlap, making it hard to read.)
Also, entering a negative value for xx will instead expand the character spacing by xx pixels.

For example, you can easily create decorative borders in prose text.
When condensed, it looks like this.
■　Change the line spacing in a message
Use \space[xx].
Changes subsequent line spacing to xx pixels.
This can be useful when the line spacing is too wide due to small font size, etc.
Like \-[xx], making it too small can make things hard to read.
(Negative values are also possible. Going too far will cause lines to overlap.)

For example, use it when adjusting display position. It can also be used to create decorative borders.
The height gets condensed.
■　Move the display coordinate of text
Use \mx[xx], \my[yy], \ax[xx], \ay[yy].
(\ax[xx] and \ay[yy] were added from Ver2.00.)

With \mx[xx] and \my[yy], the position of subsequently displayed text is
shifted by xx pixels in the X direction and yy pixels in the Y direction. (Relative specification)

With \ax[xx] and \ay[yy], the position of subsequently displayed text is
forcibly set to xx pixels in the X direction and yy pixels in the Y direction. (Absolute specification)

Using this, you can display overlapping text with a single string picture.
Note that the next character after this (the reference character) will be displayed
at the position determined by this command, and subsequent characters
continue normally after the reference character.
Also, the reference point for coordinate movement is
system variables 1 and 2 "Message Window X coordinate (Y coordinate)".
(For picture text display, the reference coordinate when displaying that picture should be used.)
The effective range of \mx[xx] and \ax[xx] is the line in which they are used. They are cancelled at a newline.
The effective range of \my[yy] and \ay[yy] is the entire message.

For example, you can add a decorative shadow to title text.
Depending on the method, you could even create a "scrambled word quiz!"

■　Add ruby (furigana) to text
Use \r["text", "ruby"].
Enter the text you want to annotate in "text", and the reading in "ruby".
The text specified in "ruby" will be displayed above "text".
Use it for difficult-to-read characters.
Note that the ruby color is specified by the color of data slot 13 in System Database
Type 12 "Text Color".

For example, when making a game for younger players, you need to add readings to kanji.
Even in games for adults, ruby is useful when using kanji with difficult readings,
or when you want a word read in a non-standard way.
However, before adding ruby, you should consider whether it is even necessary
to use difficult kanji in the first place. If the reading feels off to the player,
there is a risk of "breaking immersion"...
Even difficult characters become readable with this.
What are you hesitating for?
■　Display an icon inside a message
First, refer to "◆ I want to add an icon to item names".
But similar things can be done during message display too.
Use \i[XXX] or \img["address"].
XXX is a number (000～999), and "address" is the address of the image you want to use (starting from Data; see image for reference).
\i[XXX] displays the icon from the file "iconXXX.png" (XXX is a number 000～999) inside the message.
If that numbered file does not exist, nothing is displayed.
With \img["address"], enter the address of the icon you want, and it will display that image inside the message.
In this case, the icon's filename does not need to be "iconXXX.png",
and it does not need to be in the "BasicData" folder.
However, this may be slower than \i[XXX].
Also, if the file specified does not exist, an error will occur.
Note that depending on the icon size, the text alignment may become uneven.
In that case, use the previously mentioned "\m[xx]" or "\my[xx]" to adjust the alignment.

As for usage: \i[XXX] is straightforwardly used when you want to display an icon in a message.
\img["address"] is very difficult to use effectively, but there are precedents for using it
to display a diagram inside the message window in a gag scene, for example.
Displaying a face graphic.

■　Display text centered or right-aligned
Use ＜C＞ and ＜R＞ respectively. (Note: use half-width ＜＞ brackets.)
To return to the initial left-alignment, use ＜L＞.
The reference for alignment is the longest line in the message.
The effective range is the line in which it is used.

For example, this allows you to create text displays resembling documents. (Should be able to look like a report.)
It can serve as an accent for report headings, epitaphs, and similar expressions.
An impromptu mock report.
Is Wolfar a science student?
■　Wait a fixed time during message display before the next display
Use \.
When text display reaches this character, it waits 0.25 seconds there.
For example, using \. four times in a row will pause there for 1 second.

For example, it can be used when announcing lottery winning numbers one digit at a time, for countdowns,
or when counting something.
3, 2, 1, …what comes next? Wait!
■　Pause message display partway through
Use \!
When text display reaches this character, the message display temporarily stops.
To display the remaining text, the player must press a key.
(The pause cursor does not appear until the first stop, but it appears after that even without stopping.)

For example, in long monologues or sound novel-style reading material,
you may need to stop the message at paragraph breaks to control pacing.
Take it easy now.
How many people even know about this?
■　Display the entire message at once
Use \>
Text that comes after this is displayed instantly.
This effect is more pronounced when the text display speed (changed with system variable 9 or \sp[xx] described below) is slow.
To display only part of the message instantly, use \<.
Only the part enclosed by \> and \< is displayed instantly.

For example, it can be used in horror effects... might give you a scare!
"Boo!!" Don't use this on people with weak hearts!
■　Erase a message without waiting for key input
Use \^
When text display reaches this character, it behaves as if a key was pressed during message display, and the message disappears immediately.
However, without something like \., the message containing this will be erased immediately and cannot be read. Choose when to use it carefully!
Also, combining this with the \> described above will completely skip that message.
It can also be used for skipping troublesome messages.
(However, if used incorrectly, important messages can be completely skipped
and become unreadable, so be careful.)

For example, you can express a rapid-fire machine-gun talking style for a character.
Of course, this style makes it likely that the player cannot keep up with the messages,
so avoid this kind of expression when the character is saying something important.
A flood of text! (They probably won't even read it.)
Note: in this example, once all text is displayed, the window immediately closes.

【Addendum】 Please read this too.

■　Oh right — special characters use \ (yen sign or backslash)...
Yes, \ is used in these ways, so you cannot display it by just typing it normally.
When you want to use this character, type "\\" — two of them together.


★Note!
Because of this specification, if you use \ in a file address,
it may not be processed correctly.
Therefore, when specifying addresses and similar,
it is recommended to use "/" (forward slash) (especially from Ver1.16 onwards).
From the sample game — perfect for explaining special characters.
■　So what is \n then?
Looking at the Event Command screen, you may sometimes see a string like "\n".
This is actually also a type of special character representing a newline.
However, typing this inside a message does not create a newline, so be careful.
When you press Enter in a Show Message box and add a newline, it is replaced and displayed as \n internally in the command.
This. Typing "\n" does not create a newline.
