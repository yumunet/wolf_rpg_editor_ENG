【Use a Different Map Chip Asset in Your Game】

[Goal]

This article introduces how to use map chip assets different from the default ones.
This tutorial covers how to prepare base chips and auto-tiles, import them, and make them usable.
Success means the registered map chips can be used when creating new maps.
By the way, using the assets from this tutorial you can create a map like the one shown on the right.
Additionally, this article also covers tileset files.
The creator of this article is quite a perfectionist (lol).
For the techniques used here, see
"◆Draw Auto-Tile Borders Freely".

【Steps】

[1]　First, prepare the map chip assets
　Download the necessary chip files from here.
　Placing the downloaded chips in the "MapChip" folder makes registration more convenient.
　(Note: The placement of Sample.tile is flexible — put it wherever is easy for you to manage.)

　Note that the assets in this tutorial are designed for a resolution of 320×240.
　Set the resolution in "Game Basic Settings".
Use your browser's "Save Image" function to save them.
The tileset file is available here.
Please use "Save Link As".
[2]　To register
　Click the icon to open the map chip settings screen.
　Then click on an empty number slot.
[3]　Register a name.
　The text entered in the name field is used to identify the chip set.
　Here, let's register it with the name "test".
Right here.
[4]　Register the base chip you want to register.
　Register the base chip in the "Base Tileset File" field.
　Here, let's register "Base.png".
　Press the "Load" button next to the blank field, and a dialog will appear,
　from which you can select the map chip you want to use.
　If the selected map chip is displayed on the right side of the screen, you're done.
Did the contents of Base.png appear on the right?
[5]　Register the auto-tiles.
　Here, register "Carpet.png", "Grass.png", and "Water.png".
　Register the auto-tiles you want to use in the fields below "Auto-tile File".
　The registration method is the same as for base chips.
　If part of the selected auto-tile appears at the top of the right side of the screen, you're done.
Did the selected auto-tile appear on the right?
※Don't forget passability settings
　Without proper passability settings, the game won't behave as expected.
　For details and how to configure passability settings, see "◆Change Map Chip Passability Settings".
　For this tutorial, set them as shown in the screen here and you'll be fine.
Configure passability like this.
Direction settings can be left as-is.
[6]　Can you use the registered map chips?
　Once registration is complete, let's try using them.
　
　Create a new map (click the icon),
　and open the "Tileset" list from the dialog that appears.
　If the name "test" you registered earlier appears here, you've succeeded.
　Select it and click OK.
　You can now draw maps using the registered "test" chips.
If it looks like this, you're done! You can now use them as shown on the right.

[Side Note]

【A】　Tileset files (.tile files) let you do everything at once!
　In WOLF RPG Editor, the settings configured in the map chip settings can be exported as a tileset file.
　It contains the name, base chips, auto-tiles, passability settings, and more,
　and loading the file will automatically load all of these items.
　For passability settings, see "◆Change Map Chip Passability Settings".
　However, when loading this file, the chips used must be placed in the "MapChip" folder in advance.
　
　This time, let's load "Sample.tile".
　Click "Load" for the tileset on screen and select the tileset file above.
　This will load the name, base chips, auto-tiles, passability settings, etc. all at once.
　
　Also, clicking "Save (Single)" for the tileset exports the settings information
　(name, base chips, auto-tiles, passability settings, etc.) of the currently selected map
　as a tileset file in .tile format.
You can load it here.
【B】　Number of auto-tiles that can be loaded
　If WOLF RPG Editor is Ver.2 or later, you can register up to 15 auto-tiles.
　However, only 8 registration slots are displayed at a time.
　To register auto-tiles from slot 8 onward, click "2" next to "Auto-tile File".
　This switches the available auto-tile slots to 8–15 (click "1" to switch back to 1–7).
　
　※Note!
　Slot 0 cannot be registered. (It is marked with ×.)
　This chip is reserved for chip erasing (the eraser tool).
