【Adding an icon to an item name】

[Goal]

We will add an icon to an item's name.
Here, we will use the sample game item "Medicine Bottle" as an example to add an icon.

　※Related topics
　I want to add more item types (items)
　I want to add more item types (weapons)
　I want to add more item types (armor)

[1]
First, click to open the User Database.
For this example we will use the "2:item" entry, so please click "2:item".

[2]
Next, select the item you want to add an icon to.
Click the item name in the data ID list. (Right column of the [1] image)

　→ If you are unsure, refer to steps [1]–[4] of I want to add more item types (items).

[3]
In the red-bordered area of the image,
add the special character \i[number] that directly displays an icon to the item name.

※
The icon displayed changes depending on the number you put inside the [].
To find out which number corresponds to which icon, refer to the "BasicData" folder inside the "Data" folder.

Important!
In ver1.20, there is a bug where icons cannot be used every 112 icons starting from number 78.
In ver1.20, do not use icons with numbers such as "78", "190", "302", etc. Alternatively, use ver1.30 or later.

[4]
Once you have added it, click the Update button or the OK button.
Your changes have now been saved.

[5]Test Play
Finally, let's confirm that the icon displays correctly in the game.
If the icon appears before the item name as shown in the image, you have succeeded.

If the icon is not displaying correctly...
Check whether the \ or numbers are using full-width characters, and confirm you have not forgotten the [].




[Side Note] "weapon" and "armor"
The example above used items, but
you can also add icons to weapons and armor.

At step [2], select "3:weapon" or "4:armor".
After that, just add \i[number] following the same procedure as [3].
