【I want to change a character's name】

[Goal]
Change a character's name using an event.
Version used: [2.x series]

>>> Advanced <<<
◆ Change a character's name to one entered via keyboard input



[1]
　• First, use the Map Selector to switch to the "map where you want to create the event".

　• Create an event somewhere suitable and set it to the chicken image.

　For steps, refer to [1]–[5] of ◆ An event launched by the Confirm key.




















[2]
　Once created, click
　"■ Show Command Input Window ■"
　at the bottom right of the Event window.

1. When the "Insert Event Command" window appears,
　click I Common Event to select it.

2. Confirm that "Call by Common Event Name" (circled)
　is selected.
　If the circle next to the text has a dot in it,
　it is selected.
　※ If it is not selected,
　click it to select it.

3. Clicking the area circled by box 3 will display
　a list of available Common Events.
　Since this is a character name change event,
　select "○ Change Player Character Info".




















[3]
　1. Select the character you want to change.
　Check that the white checkbox is unchecked.
　If it is checked, click the checkbox to uncheck it.
　Clicking the field will display a character name list —
　select the character you want to change.

　2. For "What to Set?", select "0: Character Name".
　Check the checkbox just like in step 1.
　Since we are changing the character's name,
　select "0: Character Name".

　3. Enter the new character name in the "New String" field.
　In this example, we are changing it to "MainHeroPerson".

　4. Finally, left-click Input and you are done.

　The process to change the character's name is now complete!





















[4] Test Play
　In this example:
　Target Player Character: "Hero"
　What to Set?: "Character Name"
　New String: "MainHeroPerson"

　If you can confirm in the menu screen
　that the name has changed as shown in the image,
　you have succeeded.

　※ If something goes wrong...
　Check whether the input fields and
　target were set correctly
　when configuring in [3].

　Great work! You're done.
