【I want to add a wait time】

[Goal]
　Use the wait process to create a wait time.

　Inside "Sample Map A" in the sample game,
　create a chicken event that says "Ko-ke-ria～nnu" when spoken to,
　waits about 2 seconds, and then
　says "Rianne!".

[1]　
　・First, use the Map Selector to switch to "Sample Map A".

　・Create an event at a suitable location and
 　set the chicken image.
　　（Place it at the position shown in the image on the right）


 → If you are unsure, refer to
　  ◆I want to create a new event's
　　steps [1]～[5].

[2]　
　・Add a Show Message process so the chicken says "Ko-ke-ria～nnu♪".


 → If you are unsure, refer to
　  ◆I want to create a new event's
　　steps [7]～[10].
[3]　
　・Open the Insert Event Command window and
　　click the "Other 1" tab.
[4]
　・Select wait in the lower left,
　　and enter 120 in the text box labeled "frame".

　・After clicking the input button in the lower right,
　　confirm that "■ Wait: 120 frames" has been
　　added below the message "Ko-ke-ria～nnu♪".

　　　
[5]
　・Add a Show Message process so the chicken says "Rianne!".
[6]　
　The process is complete.
　
　　1: Lower left, Trigger Condition is "Execute on Confirm key"
　　2: Right side, the Event Command display shows:
　　　■ Message: Ko-ke-ria～nnu♪
　　　■ Wait: 120 frames
　　　■ Message: Rianne!

　Please confirm it displays as shown above.
[7]　
　After clicking "Save (Entire Map)",
　click "Test Play".
[8]　
　Talk to the chicken and
　if there is a wait time between when
　"Ko-ke-ria～nnu♪" is displayed and
　"Rianne!" is displayed,
　you succeeded.
　Good work!
　[Side Note]
　・The unit of wait is not time but "frames".
　　The time per frame varies depending on settings and the play environment.
　　When creating wait times in time units,
　　it may be better to obtain time from system variables.
　　→【WOLF RPG Editor description book】：[System Variable List]

　　The "Game Processing FPS" in "Game Basic Settings"
　　lets you select the base FPS (Frames Per Second = how many frames are processed per second).
　　The default is 60 FPS.
　　In this example, we applied a 120-frame wait,
　　which should result in approximately 2 seconds of wait time.
　　You can confirm the FPS during Test Play in the title bar.
　　

　・When waiting for player input,
　　use "Key Input" with "Wait for key press",
　　

　　and when waiting for an event's action to finish,
　　using "Wait Until Done" is convenient.
