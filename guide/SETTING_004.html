【I want to change the message-wait image】

　[Goal]

　In the message window displayed by the Event Command "Show Message",
　a red downward-pointing triangle image is shown at the center-bottom.
　This is called the "input-wait pause image",
　and changing this image can easily
　alter the atmosphere of your game.

　This time, we will learn how to change system images by replacing
　the input-wait pause image with a prepared image.

　　

　This image is available for use as an asset under the basic usage rules.
　If you wish to use it, the details of the basic asset usage rules are described in
　the article "What are the rules for using assets?"
　— please check it.

　【Steps】
[1] Prepare the asset

　Right-click the image "Text_Pause01.png" (found in the [Goal] section above) and
　perform "Save As". Save it to the
　"SystemFile" folder inside the Data folder.
[2] Open the System DB

　Clicking System Database will display
　a window like the one on the right.
[3] Select the type to edit

　① Right after the window opens, the "Type" is set to "Map Settings".
　　Click "10: Window Image".
　② There is a data list in the middle of the window.
　　Select "○ Input-Wait Pause Image" from this list.
[3] Set the image

　　Next, set the image to be used for the window.

　① Press the [File] button to the right of the image filename;
　　this opens the folder containing assets.
　② Select "Text_Pause01.png", and press the
　③ "OK" button at the bottom-right of the window.
　　"SystemFile/Text_Pause01.png" has now been entered as the image filename.
　④ Next, in the item just below the image filename, "Vertical Division Count (some only)",
　　enter 6 (it is already set to 6 in the initial settings),
　⑤ then press the "OK" or "Update" button at the bottom-right to save.
[4] Verify

　Let's do a Test Play and look at the center-bottom of the message window.
　Before the change, a small red triangle was moving up and down, but
　after the change you should see a silver triangle spinning horizontally.
　Great work!
　[Side Note]
1. About the vertical division count

· In WOLF RPG Editor, some images can be animated by dividing
　a single image into frames for the animation
　and displaying them sequentially — this is the "division count" setting.
　The image in the folder is not actually cut up;
　it is only divided pseudographically when displayed,
　so please be reassured.
· The input-wait pause image can only be divided vertically for animation.
　If the "Vertical Division Count (some only)" setting is 2 or more,
　the image's height is divided equally by that number. The width does not change.
　If the vertical division count is 1 or less, no animation occurs.
　Looking at the default red downward-pointing triangle image "Text_Pause.png",
　you can see six triangles arranged vertically.
　Since the vertical division count was set to 6, these triangles are
　displayed one frame at a time.
2. Changing the message window background

　This time we changed the pause cursor displayed at the bottom of the message window,
　but you can also change the image that serves as the background
　of the message window.
　This can be changed in User Database Type 16 "Image/Audio Settings",
　Item 2 "Message Window Base Image".

　　　

　This image is also available for use as an asset under the basic rules.
　Give it a try if you have the time!
