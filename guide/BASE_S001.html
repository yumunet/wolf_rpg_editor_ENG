[I want to set the player character's walking graphic]

[Goal]
　We will change the walking graphic of Wolfar (the blue character), the player character in the sample game, to a cook.

 → 

　※This page covers setting the player character's walking graphic at game start.
　　 To change the player character's walking graphic during the game,
　　 see "I want to change a character's title and image mid-game".

[1]
　Click the Changeable Database
　to open the window shown on the right.
　　
[2]
　Confirm that the "Type" shows "Player Character Status".

　(If not, click "0: Player Character Status".)
[3]　
　Looking to the right, you will see a list like the one shown.
　Wolfar should already be selected.

　Click the "File" button to the right of
　"Walking Graphic Image".
[4]　
　The "Select File" window will appear.
　Select "Cook.png" from the list on the left.


※To revert, select "Wolfarl.png".

 ※Use the [CharaChip] box in the top-left
  　to choose which folder to load from.
　　
[5]　
　Run Test Play and
　Wolfar's (the blue character's)
　walking graphic has changed
　to the cook.


　The opening dialogue is still the same,
　which looks a bit odd, but regardless —
　the player character's walking graphic
　has been changed successfully!
