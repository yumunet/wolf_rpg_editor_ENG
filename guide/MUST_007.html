[I want to set the game's title name]

[Goal]
　Set the title displayed in the game window.

[1]　Click "Special Settings" in the menu,
　　　then click "Game Basic Settings".
[2]　A new window called "Game Basic Settings" will appear.
The content of the "Game Name" field highlighted in red
is the text that will be displayed as the title.
By default it shows "Wolfar's Sample Game".
[3]　Now try changing it to a title you like.
　You can enter up to 19 full-width characters
　or 31 half-width characters.
　When done, click the OK button in the Basic Settings window.
If the message 'Game information saved (your title)' appears,
　you're done.
[4]　Click to start Test Play,
then press F8 on the keyboard, or launch "Game.exe"
from the WOLF RPG Editor folder (see image).
[5]　Confirm that the game title has been set correctly.
※ Save data for games made with WOLF RPG Editor is managed using the "Title"
　　set on this page to identify which game it belongs to.
　　If you publish a game called 'sample' and then rename it to 'sample2',
　　save data from 'sample' (using the default filename "SaveDataXX.sav")
　　will no longer be recognized by 'sample2'. Please be careful.
