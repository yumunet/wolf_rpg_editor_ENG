【I want to put my work-in-progress data on a USB drive and work on it away from home】

[Goal]
　Copy the data you are editing to a USB drive
　so you can edit it on another computer as well.

※ This description assumes you are already able to perform
　 basic operations such as copying files and moving them.

[1]　
　If you are currently using the editor,
　close the editor first.
[2]　
　Copy the entire "WOLF_RPG_Editor" folder.
[3]　
　Open the USB drive you are carrying
　from My Computer or equivalent.

　If necessary, open a suitable location where
　WOLF RPG Editor can be placed.

　Perform a "Paste" here
　to paste the WOLF_RPG_Editor folder
　that you copied in [2].
[4]　
　If you can launch Editor.exe inside
　the copied WOLF_RPG_Editor folder,
　you should be able to launch it
　from another computer as well.

　Great work!
[Side Note]
　· Even when launched from a USB drive, operations such as saving and file references work as normal.

　· If copying the entire WOLF RPG Editor folder is too large and takes too long,
　　copying only the changed parts is also fine.
　　For maps and various Databases and Common Events, copy the "Data" folder;
　　for Game Basic Settings, copy "Game.dat"; for save data, copy the "Save" folder —
　　overwriting the destination with each of these will update them.

　· For even more efficiency, it is also possible to move only specific files within the Data folder.
　　Example: Only map events were edited, so copy only the "MapData" folder, etc.

　　When copying only part of the data, it is safer to also copy the "BasicData" folder.
　　(The BasicData folder contains many items related to other data,
　　　such as the contents of each Database, the map tree structure, and Common Events.)
