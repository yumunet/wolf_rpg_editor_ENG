【Use RPG Maker-Sized Assets in WOLF RPG Editor】

[Goal]
On the web, many assets for game creation are distributed, and most of them conform to "RPG Maker" specifications.
Some of these assets differ from WOLF RPG Editor's specifications.
This article introduces how to convert such assets so they can be used in WOLF RPG Editor.
Here, we will use the method of using "tkool2WOLF+", distributed on the official page.

●About the images used
The images used in this tutorial are character chips and map chips for RPG Maker 2000,
distributed by "First Seed Material (distributor: REFMAP)".
They are available under "DOWNLOADS" on the page, in "character chip" and "map chip" sections.
Please obtain the assets from there.
★Note!
　When using assets, always read and comply with the asset's terms of use.
Also see "◆What are the rules for using assets?".

【Steps】

[1]
Download "tkool2WOLF+" from the following location on the official site.
The download page is here.
On the official site top page, the link is circled here
[2]
Launch tkool2WOLF+.
A window like the one in the figure will appear.
This will appear; the checkboxes at the bottom are explained in the [Side Note]
[3]
Prepare the image you want to convert (compatible: character chips and map chips/auto-tiles for the 200x, XP, and VX series).
Select the image from "Open File" (or drag and drop the image).
[4]
tkool2WOLF+ will determine the image's specifications and select the necessary process for you.
If it cannot auto-detect, it will ask you which process to perform — select the one you want.
For example, like this
[5]
Click "Start" and a folder will be created at the original image's location, with the processed image output inside.
After that, rename the file as desired and use it in your game.
Done!
If you click "No" here,
the tool closes.

[Side Note]

【A】About other features of tkool2WOLF+: Specifying transparent color
As an option, you can also "specify a transparent color".

- First, check the "Specify transparent color" checkbox.

- Set the image you want to process using the above method and start the process!

- A window like the one in the figure will appear.

- Select the color you want to make transparent. You can also pick from the image.

- The color you choose here will become a gray checkerboard pattern. If the result is as expected, click "OK" to finish!


For general transparency methods, see "◆I want to make images transparent".
Before processing — the blue-green area will become transparent.

If this is transparent, you're done.
【B】About other features of tkool2WOLF+: Map merging
Similarly, you can also "merge maps" as another option. (Described below.)

- Selecting and starting the conversion process for map chips is the same, but
　check "Merge maps" before starting.

- After starting the conversion process, a dialog appears showing the converted map chips arranged vertically.
　The result of subsequent processes will also be shown here, allowing you to verify before saving.

- At this point, you can change the order of chips, add chips to merge, or remove chips you don't need.

- You can select the resolution (320×240 or 640×480) for the output chip.
　Choose the one that matches your intended specification.

- To save the merged map chip, click "Save".

- When finished, click "Exit" to end the process.

There is also a separate tool "MapMapper" derived from the above option.
Use it when you want to merge already WOLF RPG Editor-compatible map chips into one.
