【An Event That Remembers How Many Times You've Talked】

[Goal]
When you talk to the chicken event, it will display "This is the ○○th time we've talked!", and
we want to create an event where new dialogue is added after talking 10 times.
　→　

[1]・First, use the Map Selector to switch to "Sample Map A".
　
　・Create a new event somewhere on the map.

　・Create an event somewhere and
　set the chicken image.
　　（In this example, place it at the position shown in the image on the right）

　→ If you're not sure how, refer to steps [1]～[3] of
◆I want to create a new event.
[2]Click the
　　 "■ Show Command Window ■" button
　　 in the bottom-right of the Event window.
[3]Switch to the Variable Operations tab and
　　 make changes as shown in the image.
　　 Then click input.
　　 This will track the number of times you've talked!

　　 Here we use "This Event: Self0"
　　 as the counter for how many times you've talked.
　　 Each time this event is triggered, Self Variable 0
　　 is incremented by 1 (+= 1), so
　　 the "number of times talked" also increases
　　 by 1 each time.
[4] Following steps [7]～[10] from
　　 ◆I want to create a new event,
　　 enter the text "This is the \self[0]th time we've talked!".

　　 \self[0] is a special character that represents
　　 "the number stored in Self Variable 0 of this event".
　　 At this point, you have an event that says
　　 "This is the ○th time we've talked!" for each conversation.
　　 Next, we'll make the dialogue change
　　 after 10 or more conversations.
[5]Next, click the "Conditional Branch" tab.
　　 Then configure the settings as shown in the image.
　　 This checks whether the player
　　 has talked 10 or more times.
[6]Move the cursor to the position shown in the image.
[7]Enter the dialogue for after 10 or more conversations.
　　 Enter "We've already talked \self[0] times now!".
[8]If the final result looks like the image,
　　 you've succeeded.
　　 Please do a Test Play to confirm.
　　 Great work!
　　
　　 If something isn't working, check the following symptoms and causes.

　　 ・The number in the dialogue doesn't increase.
　　┣Check the Variable Operations in [3].
　　┃　　 Is the variable set to This Event: Self0 and the operator set to +=?
　　┗Check that the special character in the [4] message display is \self[0].

　　 ・The dialogue doesn't change even after talking 10+ times.
　　┣Check the Conditional Branch in [5].
　　┃　　 Is the variable set to This Event: Self0?
　　┗Also check [6] and [7] to confirm that the dialogue is inside the Conditional Branch.
[Side Note]
What's this "\self[0]" thing?!
This is called a special character — a character that can be used in text and other places within WOLF RPG Editor.
That's why we used a special character to display the number of times talked in this tutorial.
There are many special characters. Here are some examples:

・\self[X] = Self Variable X
・\s[XXX] = String Variable XXX　etc.

These can be looked up at any time in the WOLF RPG Editor help under "Variable Call Values / Special Character List".
Use these to enhance your game creation!
