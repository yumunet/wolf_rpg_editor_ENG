[I want to add more skill (magic) types]

[Goal]
Add a new skill (magic). We will create a skill that combines HP healing and a guard effect.


[1]
Click the User Database
to open the window shown on the right.
[2]
Confirm that the "Type" shows "Skill".

(If not, click "0: Skill".)
[3]
Click an empty "Data" slot.
[4]
Fill in the fields on "Page 1".

Fields entered here:

· Skill name
· Description
· SP cost

· Base effect amount (at least 100 HP healed)
· Mind influence (heal amount increases by 1.5× mind stat)
· Damage variance (heal value varies by ±20%)

· Message on use
· Priority/delay action option

※ Reference values are based on
　the sample game's "Heal" and "Guard" skills.

※ For fields you're unsure about,
　copy values from the sample game's skills.
→
[5]
Fill in the fields on "Page 2".

Fields entered here:

· Status condition to apply
· Status condition to cure

· Effect on enemy's defense/mind
→

[6]
Click the Update button or OK button.
The new skill "Wolf Change" has been saved.
