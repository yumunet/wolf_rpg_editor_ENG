【I want to make the text in a message larger】

[Goal]
　Create a message that displays normal text, large text, and small text.

　Here, inside "Sample Map A" in the sample game,
　create a chicken that first says "chicken medium text" in normal size,
　then "chicken large text!" in large text,
　and finally "chicken small text…" in small text.

　

[1]　
　・First, use the Map Selector to switch to "Sample Map A".

　・Create an event at a suitable location and
 　set the chicken image.
　　（Place it at the position shown in the image on the right）


 → If you are unsure, refer to
　  ◆I want to create a new event's
　　steps [1]～[5].

[2]　
　・Add a Show Message process so the chicken says "chicken medium text".


 → If you are unsure, refer to
　  ◆I want to create a new event's
　　steps [7]～[10].
[3]
　Similarly to before, this time enter
　"chicken large text!".
　
　Then, to change the text size,
　enter "\f[32]" at the beginning of the line.
　"\f" means "change the font size",
　and "[32]" specifies that the size is 32.
　
[4]
　Next, display it small:
　"chicken small text…".
　
　Making text display smaller or larger works the same way —
　the only difference is whether
　"the specified size is larger or smaller".
　
　So, enter "\f[8]" at the beginning of the line.
　
[5]
　The chicken event that speaks three messages with different text sizes is now complete.
　Does the completed event settings look like the following?
　
　1: Lower left, Trigger Condition is "Execute on Confirm key"
　2: Right side, the Event Command display shows:
　　　■Message: chicken normal text
　　　■Message: \f[32]chicken large text!
　　　■Message: \f[8]chicken small text…
　　　■
[6]
　Now save the map and
　test the event you created.

　After clicking "Save (Entire Map)",
　click "Test Play".
[7]　
　If the message "chicken large text!" is displayed in large text
　as shown in the image on the right,
　the large text display was successful.

　Good work!
　


[Side Note]
　The size is relative to the default value.
　Basically, in the initial state, the default size is "12".

　So, "\f[12]" gives the default (unspecified) size,
　values of 12 or less such as "\f[10]" give a smaller size,
　and values of 12 or more such as "\f[14]" give a larger size.

　By the way, this value is set by system variable number 8, "Base Font Size",
　and you can change the text size in the unspecified state by modifying it with Variable Operations.
