【EV_011: ◆I Want to Change the Movement Speed of the Player Character or an Event】

　[Goal]

　Changing movement speed can be used for various staging effects.
　For example, movement speed slowing down when leaving a town and entering a vast field,
　movement speed dropping in swampy or difficult-to-traverse terrain,
　or a trap that makes you move too fast to stop when stepped on.

　This time, we will configure the sample game's dungeon so that
　"your movement slows when you enter, and returns to normal when you go back to town",
　changing the player character's movement speed.
　To change movement speed only inside the dungeon,
　we not only need to "slow the movement speed upon entering",
　but also "restore the movement speed upon exiting".
　These two steps are added to the enter-dungeon event and the exit-dungeon event respectively.
　Changing movement speed basically requires these two steps as a paired set.

　【Steps】
　■ Slowing Movement Speed When Entering the Dungeon
[1] Open the event

　Open "Sample Map A" and open the window for
　event "02: Dungeon Door" placed at the dungeon entrance.
　Move the cursor to the Place Transfer command inside the Conditional Branch
　and click to select that location.
[2] Open the Event Command Input window

　Click "■ Show Command Window ■"
　in the bottom-right of the Event window.

　Next, switch the tab to "H Event Control",
　and click "■ Character Movement"
　in the upper-right of the window.
[3] Character Movement

　① Here you can configure the character's movement.
　　 Let's set the player character's movement speed to speed 2.
　　 From the dropdown list at the upper-left of the window,
　　 select "player character".
　② Also, in the speed/frequency settings at the center-bottom of the window,
　　 select 2 from the dropdown list,
　　 then press the button below labeled "Set Movement Speed".
　③ When done, press the "OK" button at the bottom-right.
　■ Restoring Movement Speed When Exiting the Dungeon
[1] Open the event

　Open the "Dungeon" map and open the window for
　event "00: Entrance Place Transfer" placed at the dungeon exit.
　Move the cursor to the Place Transfer command
　and click to select that location.
[2] Enter the Event Command

　Just like the "Slowing Movement Speed When Entering the Dungeon" step above,
　this time set the player character's movement speed to speed 3.

　Click "■ Show Command Window ■"
　in the bottom-right of the Event window.

　Next, switch the tab to "H Event Control",
　and click "■ Character Movement"
　in the upper-right of the window.
[3] Character Movement

　① From the dropdown list at the upper-left of the window,
　　 select "player character".
　② In the speed/frequency settings,
　　 select 3 from the dropdown list,
　　 then press the button below labeled "Set Movement Speed".
　③ When done, press the "OK" button at the bottom-right.
[4] Confirm the event

　The event content has been entered.
　Let's do a Test Play.

　Try going in and out of the dungeon. If the movement speed
　is slower only inside the dungeon, you've succeeded.
　Great work!
　[Side Note]
　Once changed, the movement speed of the player character or event will remain at that speed
　until changed again — it will not automatically revert
　when switching maps, saving/loading, etc. Therefore, if you temporarily change the movement speed,
　you need to add an event to restore it
　at the point when the temporary speed change is no longer needed.

　The initial movement speed of an event can also be set in that event's "Movement Route".
　If movement speed is changed by an Event Command, that command takes priority.

　Also, as a slightly intermediate technique, applying the content from this tutorial
　allows you to create a key-dash feature where movement speed increases
　only while a specific key is held down.
　For details, please refer to the article "A Feature That Increases Player Character Movement Speed with a Specific Key (Dash Feature)".
