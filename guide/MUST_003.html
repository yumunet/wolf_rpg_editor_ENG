[I want to move the player character to a different map]

[Goal]
　We will create an invisible event next to the barrel in Sample Map A
　that teleports the player to Sample Map B when touched.


Walk to the barrel in Sample Map A...		  → .
　
...and you are transported to in front of the goddess statue in Sample Map B.

　
　

[1]　
　Open the Map Selector and
　select "1: Sample Map A".
　
[2]
　Switch to the Event Layer ,
　then right-click next to the barrel near the start position.
　A menu will appear as shown.

　Select "Create/Edit Event".
[3]　
　The Event Settings window
　shown on the right will appear.

　
[4]　
Set the Trigger Condition to "Player Touch".

　
[5]　
　Click "■ Show Command Input Window ■"
　to open the "Insert Event Command" window.

　Then click the "Location Transfer" tab at the top.
[6]
　The window shown on the right
　will appear. Click
　"Specify while viewing destination".
[7]　
　The "Select Location Transfer Destination"
　window will appear.
　Click "2: Sample Map B" in the left panel
　to switch maps.

　Click in front of the goddess statue
　on the right map to set
　the red-and-white cursor.

　Click "OK" when done.


　※Middle-click and drag on the right
　　map to scroll the view.
[8]
　You can see that the destination map
　and destination position now have values.

　Click "Input" to register the event command.
[9]
　When done, click "Save (Entire Map)"
　to save, then start Test Play.
[10]　
　Start the game and walk toward
　the barrel to the right of the start point...
[11]　
　After a brief pause, you are transported to
　in front of the goddess statue in Sample Map B!

　The location transfer event connecting
　two maps is complete!





[Side Note]
　I want to move to the "Test" map created in
　[How do I create a new map?]!


　Using the same method, create a location transfer event
　from the barrel area in "Sample Map A"
　to the "Test" map.

　However, if you only have an event going from
　"Sample Map A" to "Test",
　you won't be able to return to "Sample Map A".
　You also need to create a
　return event from "Test" back to "Sample Map A".

　To properly connect two maps, you need
　two location transfer events: one to go and one to return.
