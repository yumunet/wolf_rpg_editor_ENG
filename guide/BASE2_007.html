【I want to configure detailed game system settings】

　[Goal]

　In the User Database of the Basic System, you can change the terminology and images used in various
　places throughout the game.
　This time, we will configure settings to change "item" in the menu to "item",
　change the window image, and hide the enemy HP gauge,
　to understand what kinds of things can be configured and how.

　【Steps】
　■ Changing "item" in the menu to "item"
[1] Open the User Database

　Clicking the User Database button will
　display a window like the one on the right.
[2] Select the type to edit

　When the window first opens, the "Type" will be set to "skill".
　Please click "15: Terminology Settings".
[3] Edit the item

　In the input field for item 0 "[Menu] item" on "Page 1",
　please enter "item".
　Once entered, press the Update button or OK button.
[4] Test Play

　Let's do a Test Play.
　The name of the "item" entry in the menu has
　changed to "item". Only the name has changed;
　the functionality is completely the same.
　■ Changing the window image
[1] Open the User Database

　Clicking the User Database button will
　display a window like the one on the right.
[2] Select the type to edit

　When the window first opens, the "Type" will be set to "skill".
　Please click "16: Image/Sound Settings".
[3] Edit the item

　Press the "File" button to the right of item 0 "Window Image File" on "Page 1"
　to browse the folder containing images.
　Select "WindowBase.png" from there and press the OK button.
　"SystemFile/WindowBase.png" will be entered in the input field.
　Press the Update button or OK button to save.
[4] Test Play

　Let's do a Test Play.
　When you open the menu, the window image behind the text
　should have changed.
　■ Hiding the enemy HP gauge
[1] Open the User Database

　Clicking the User Database button will
　display a window like the one on the right.
[2] Select the type to edit

　When the window first opens, the "Type" will be set to "skill".
　Please click "17: System Settings".
[3] Edit the item

　Change item 27 "[Battle] Enemy Gauge Display" on "Page 2".
　Select "[0] Do not display" from the dropdown list.
　Press the Update button or OK button to save.
[4] Test Play

　Let's do a Test Play.
　The opponent's HP gauge during battle is no longer visible.
　[Side Note]
■ About UDB15: Terminology Settings

　This time we changed the name of the menu item "item",
　but you can also change other game terminology such as
　HP, level, special skills, and money units.
　You can make various changes like changing "SP" to "MP" or "save" to "diary",
　but be careful not to make things too confusing….
■ About UDB16: Image/Sound Settings

　This time we changed the window image file here.
　Here you can change frequently used sound effects and image files,
　such as sound effects that play on Cancel or Save.
　Sound effects are found not only in the SystemFile folder that opens
　when you press the File button, but also in the SE folder.
　You can open it by selecting the "SE" folder from the dropdown list
　in the upper left of this window.
■ About UDB17: System Settings

　What can be done with this Database can be broadly divided into
　"menu settings" and "battle settings".

・The menu commands [0]～[18] and special menus allow you to
　change the menu contents, order, and add Common Events to the menu.
　When adding a Common Event to the menu,
　register it in "Special Menu (A～D) Name/Call Common" first,
　then call it from "Menu Commands".

・Items [19] and beyond cover settings common to the entire game's battles,
　such as status ailments, HP display methods, damage formulas,
　HP/SP handling after battle, incapacitation handling, and default BGM.
　(Battle settings common across the whole game.)
　(Character-specific settings are done in Changeable Database 0～5 for player characters,
　 and User Database 9～12 for enemy characters.)
　For the "increase parameter by 1/10 of value on level-up" setting,
　the tens digit serves as the base increase, and the ones digit determines the probability of +1.
　If "32" is entered, there is an 80% chance of 3, and a 20% chance of 4 (with the +1).
