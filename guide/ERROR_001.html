【Character appears cut in half / splits into 2 characters】

　[ Character image is not displaying correctly ]

[Goal]
　Change the settings so the character displays correctly.


　The character is displaying as cut in half…
　The character is splitting into 2 characters…
　refers to a state where things are displaying like this:

　　


　This occurs because the assets you prepared do not match the game settings.

[1]
　・From the editor's menu bar at the top, click "Special Settings".
　
　・Click "Game Basic Settings".
　
　
[2]
　The "Game Basic Settings" window will display.
　While you can configure various settings in this window,
　focus on only one area highlighted in purple for now.
　
　In the setting field for "Character Image Direction Type",
　select either "4-direction compatible" or "8-direction compatible".
　
　When this setting does not match the character image you prepared,
　the character will appear cut in half or split.
　
　See the description later in this article for more details;
　for now, just try changing the setting.
　
　
　■ If currently set to "8-direction compatible",
　　　　　　switch to "4-direction compatible".
　　（↑ Your character is probably appearing cut in half.）
　
　
　■ If currently set to "4-direction compatible",
　　　　　　switch to "8-direction compatible".
　　（↑ Your character is probably splitting into 2 characters.）
　
　
　After making the change, click the "OK" button at the bottom.
　
　That is all you need to do.
[3]
　Confirm with Test Play.
　Is it now displaying correctly?
　
　※ If it is still not displaying correctly, also check
　"[Aside 2] Animation Patterns" and "[Aside 3] Effects of Special Functions" further down this page.
　
　That's all. Good work!
　
　A more detailed description is appended below,
　so please check it if you are curious.


【Aside 1】 Detailed Description

　WOLF RPG Editor allows you to choose from two standards for character images (called character chips):
　"4-direction compatible" and "8-direction compatible".

　"4-direction compatible" refers to an image with drawings facing up, down, left, and right (4 directions),
　and "8-direction compatible" refers to an image that also includes upper-right, lower-right, upper-left, and lower-left in addition to those 4 directions.

　　

　RPG Maker-format character chips (such as assets from First Seed Material)
　are "4-direction compatible" character chips. They only have drawings facing the 4 main directions (up, down, left, right).
　
　Unfortunately, "8-direction compatible" character chip assets are not widely available online.


　Now, about the cause of the half-display or split display...
　
　WOLF RPG Editor splits a single image into sections when displaying a character — you probably already know that.
　The number of sections is determined not by the asset size,
　but by the "image direction type" and "animation pattern" settings.
　
　When the WOLF RPG Editor setting is "4-direction compatible", it splits the image into 3 horizontal sections.
　For "8-direction compatible", it doubles to 6 sections. (※ Both assume 3 animation patterns.)
　
　A "4-direction compatible" graphic should be split into 3 sections,
　but because the setting is "8-direction compatible", it is split into 6 sections,
　so only half of the image that should be displayed is shown.
　
　The case of splitting into 2 characters is similar: an image that should be displayed in 6 sections
　is displayed in 3 sections, resulting in 2 characters appearing.
　
　(※ We used 3 and 6 sections as examples, but if the animation pattern has 5 patterns, it becomes 5 and 10 sections.)
　
　
　Note that you cannot use both "4-direction compatible" and "8-direction compatible" images in the same game.
　Unify all assets to one standard.
　
【Aside 2】 Animation Patterns

　If the above method did not resolve the issue, the "Animation Patterns" setting may be incorrect.
　
　WOLF RPG Editor also lets you choose the character animation method.
　There are 2 types: "3 patterns" and "5 patterns".
　
　"5 patterns" provides smoother animation than "3 patterns",
　but there seem to be fewer assets available for 5 patterns.
　
　Now, if the "Character Animation Patterns" setting is incorrect,
　you will see an image that is cut in an odd way, not quite half.
　
　If that is happening,
　just like the steps in this article, select "Special Settings" → "Game Basic Settings",
　find the "Character Animation Patterns" setting in the "Game Basic Settings" window,
　and change the setting to correctly match your assets.

【Aside 3】 Effects of Special Functions

　Please check the filename of the character chip.
　Does it end with "T.png" or "TX.png"?
　　（e.g. [ RAT.png ]）
　　
　If the filename ends with [T.png] or [TX.png], a special function activates and changes the number of image sections.
　If you accidentally use such a filename without knowing this, the image will not display correctly.
　
　In this case, changing the filename will fix it.
　
　This is a rare situation, but we mention it here just in case.
