　 WOLF RPG Editor Archive Page

　This site is the archive page for the free RPG creation tool “WOLF RPG Editor” developed by SmokingWOLF.
　

◆ What's WOLF RPG Editor?

　WOLF RPG Editor (commonly called “Woditor”) is a free game creation tool that allows you to make RPGs using only English commands.

　Its main features are powerful Event Commandsstrong> that allow advanced processing without scripting, and a highly flexible “Database” system. Even without programming experience, users can build complex game systems.
　It can create not only RPGs, but also a wide range of genres such as adventure, action RPGs, and simulation/strategy games.

　Since its first public release in March 2008, thousands of games have been created and released using this editor.

● For users who want the mini EXE-only version: WolfRPGEditor_(version number)_mini.zip

◆ 関連ツール

　以下はpixiv BOOTH内で販売されている『WOLF RPGエディター』の関連ツール製品です。

【WOLF RPGエディター プロ版】(BOOTHへ/新ページを開く)

　プロ版はウディタを使い慣れた人向けの、『WOLF RPGエディター』の有料版です。
　●画面サイズやダウンロード処理の制限が解除されています。
　●負荷が大きくて使い方に注意が必要な処理が搭載されています。
　●海外展開などにも有用な機能が搭載されています。
　●より強度の高いセキュリティ機能が使えます。
　

【WOLF RPGエディター 翻訳サポートツール】(BOOTHへ/新ページを開く)

　『WOLF RPGエディター』のゲームデータを簡単に翻訳するための補助ツールです。
　●ゲームデータ内の全文字列を対訳Excelファイルとして抽出できます。
　●その対訳Excelファイルに翻訳文を書き込み、翻訳を行うと最大10言語まで同時に翻訳済みゲームデータを作成できます。
　●翻訳せずとも、ゲーム内の全文字列の置換用にも使えます（たとえば、日本語のファイル名をアルファベットのみに置き換える作業など）。
　●ゲームデータが更新された場合に備え、ゲームデータ側の変更をワンボタンで対訳Excelに反映する機能も付いています。
